## 🇬🇧 英文單字（TOEIC 600-800）
1. **efficient** (adj.) — 做事省時省力
   - *This new AI tool is super efficient for coding.*
   - 這個新的 AI 工具寫程式超級有效率。
2. **optimize** (v.) — 讓系統變更好
   - *We need to optimize the code for faster performance.*
   - 我們需要優化程式碼來提升效能。

   - *Can you help me troubleshoot this network issue?*

4. **interface** (n.) — 軟體或裝置外觀
   - *The user interface of this app is very intuitive.*
   - 這個應用程式的使用者介面非常直覺。
5. **collaborate** (v.) — 一起工作達成目標
   - *Let's collaborate on this project next week.*
   - 我們下週一起合作這個專案吧。
6. **prototype** (n.) — 產品初期模型
   - *We're building a prototype for our new IoT device.*
   - 我們正在為新的物聯網設備製作原型。
7. **leisure** (n.) — 工作外放鬆時間
   - *What do you usually do in your leisure time?*
   - 你閒暇時通常做什麼？
8. **sustainable** (adj.) — 長期可維持的
   - *Renewable energy is crucial for a sustainable future.*
   - 再生能源對永續未來至關重要。
9. **innovation** (n.) — 新想法或方法
   - *We need more innovation in the tech industry.*
   - 科技產業需要更多創新。
10. **update** (v./n.) — 提供最新資訊
   - *Don't forget to update the software regularly.*
   - 別忘了定期更新軟體。

## 📘 英文文法
### 第一類條件句 (If 句型)
表達未來可能發生的事情及其結果，條件真實或很有可能發生。
- If the system crashes, I will restart it. (如果系統當機，我就會重啟它。)
- If you finish early, we can grab coffee. (如果你早點忙完，我們可以去喝咖啡。)
- If it rains, I'll work from home tomorrow. (如果下雨，我明天就會在家工作。)
### 片語動詞 (與 up 相關)
動詞與介係詞 up 結合，產生新意義，常用於日常和工作情境。
- Can you help me set up this new server? (你能幫我設定這台新伺服器嗎？)
- I need to come up with a solution by tomorrow. (我需要明天前想出一個解決方案。)
- We need to speed up the development process. (我們需要加快開發流程。)

## 🇯🇵 日文單字（N5）

## 🗾 日文文法（N5）
### 動詞的「～ます / ～ません」形
**結構**：動詞ます形 / 動詞ません形
用於表達現在或未來會做某事，或是不會做某事，是日語最基本的禮貌體動詞。
- わたしはまいにちコーヒーをのみます。(我每天喝咖啡。)
- きょうはしごとをしません。(今天我不工作。)
### 助詞「を」
**結構**：名詞 + を + 動詞(ます形)
助詞「を」標示動詞的直接受詞，表示動作作用的對象。
- ごはんをたべます。(我吃飯。)
- にほんごをべんきょうします。(我學習日語。)
