# 2026-05-20（週三）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **optimize** (v.) — 優化，使效率提升
   - *Let's optimize this code for better performance.*
   - 我們來優化這段程式碼以提升效能吧。
2. **troubleshoot** (v.) — 排除故障，解決問題
   - *I need to troubleshoot why the IoT device isn't connecting.*
   - 我需要解決物聯網設備無法連線的問題。
3. **implement** (v.) — 執行，實施
   - *We decided to implement the new safety protocol.*
   - 我們決定實施新的安全協議。
4. **deadline** (n.) — 截止日期，最後期限
   - *The project deadline is next Friday, so let's hurry!*
   - 專案截止日是下週五，我們快點吧！
5. **feedback** (n.) — 回饋，意見
   - *Your feedback on the design is much appreciated.*
   - 你對設計的回饋意見非常寶貴。
6. **prototype** (n.) — 原型，樣本
   - *The team is testing the first prototype of the new gadget.*
   - 團隊正在測試新裝置的第一個原型。
7. **secure** (adj.) — 安全的，穩固的
   - *We need to make sure our network is secure.*
   - 我們需要確保我們的網路是安全的。
8. **collaborate** (v.) — 合作，協作
   - *It's great to collaborate with talented engineers.*
   - 能與有才華的工程師合作真是太棒了。
9. **innovation** (n.) — 創新，革新
   - *Innovation drives progress in the tech industry.*
   - 創新推動著科技產業的進步。
10. **efficient** (adj.) — 效率高的，有效的
   - *This new tool makes our workflow more efficient.*
   - 這個新工具讓我們的工作流程更有效率。

## 📘 英文文法
### 情態動詞 (Modal Verbs) - "Should" 和 "Must"
"Should" 表建議或責任，語氣較軟；"Must" 表必要或義務，語氣較強烈。
- You should check the log files regularly. (你應該定期檢查日誌檔。)
- We must finish this report before the meeting. (我們必須在會議前完成這份報告。)
- Engineers should always prioritize safety in design. (工程師在設計時應始終將安全放在首位。)
### 動名詞 (Gerunds) 作主詞或受詞
動名詞 (V-ing) 可當句子的主詞或受詞，表示一個動作或活動本身。
- Troubleshooting is a crucial part of my job. (排除故障是我工作中很關鍵的一部分。)
- I enjoy learning about new AI tools. (我喜歡學習新的 AI 工具。)
- Developing renewable energy solutions is exciting. (開發再生能源解決方案令人興奮。)

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - 私のしごとはエンジニアです。
   - 我的工作是工程師。
2. **かいしゃ** / 会社 / kaisha — 公司
   - 毎日かいしゃに行きます。
   - 每天去公司。
3. **かいぎ** / 会議 / kaigi — 會議
   - 午前にかいぎがあります。
   - 上午有會議。
4. **じかん** / 時間 / jikan — 時間
   - じかんは大切です。
   - 時間很重要。
5. **きゅうけい** / 休憩 / kyuukei — 休息
   - 少しきゅうけいしましょう。
   - 稍微休息一下吧。
6. **べんきょう** / 勉強 / benkyou — 學習
   - 日本語を毎日べんきょうします。
   - 每天學習日文。
7. **つかう** / 使う / tsukau — 使用
   - このツールをつかいますか？
   - 你用這個工具嗎？
8. **パソコン** / パソコン / pasokon — 電腦
   - 新しいパソコンがほしいです。
   - 我想要一台新電腦。
9. **かだい** / 課題 / kadai — 課題，問題
   - このかだいはむずかしいです。
   - 這個課題很難。
10. **おわる** / 終わる / owaru — 結束
   - しごとはごじにおわります。
   - 工作五點結束。

## 🗾 日文文法（N5）
### 名詞 + を + 動詞(ます形)
**結構**：名詞 + を + 動詞(ます形)
「を」是受格助詞，用於表示動詞作用的對象，後面接動詞的ます形。
- ごはんを食べます。 (我吃飯。)
- 日本語をべんきょうします。 (我學習日文。)
### ～ませんか (提議/邀請)
**結構**：動詞(ます形) + ませんか
用於禮貌地向對方提出邀請或建議，表達「要不要～？」的語氣。
- いっしょにコーヒーを飲みませんか？ (要不要一起喝咖啡？)
- 午後、会議室ではなしませんか？ (下午要不要在會議室談談？)
