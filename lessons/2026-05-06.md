## 🇬🇧 英文單字（TOEIC 600-800）
1. **implement** (v.) — 實施、執行計畫
   - *We need a good plan to implement this new feature.*
   - 我們需要一個好計畫來實施這項新功能。
2. **optimize** (v.) — 優化、最佳化系統
   - *Can you optimize the code to make it run faster?*
   - 你能優化程式碼讓它跑更快嗎？
3. **troubleshoot** (v.) — 排除故障、解決問題
   - *I spent all morning troubleshooting that network issue.*
   - 我花了一整個早上排除那個網路問題。
4. **scenario** (n.) — 情境、設想狀況
   - *Let's consider a worst-case scenario for the system.*
   - 我們來設想一下系統的最糟情境。
5. **feedback** (n.) — 回饋、意見反應
   - *Your feedback helps us improve the product a lot.*
   - 你的回饋對我們改進產品很有幫助。
6. **prototype** (n.) — 原型、樣本裝置
   - *We're building a prototype for the new IoT device.*
   - 我們正在為新的物聯網設備製作原型。
7. **seamless** (adj.) — 無縫的、流暢的
   - *The new update offers a seamless user experience.*
   - 新的更新提供了流暢無縫的使用者體驗。
8. **innovative** (adj.) — 創新的、革新的
   - *This innovative solution will save us a lot of energy.*
   - 這個創新的解決方案會幫我們省下很多能源。
9. **integrate** (v.) — 整合、結合不同部分
   - *How do we integrate this module with the existing system?*
   - 我們如何將這個模組與現有系統整合？
10. **intuitive** (adj.) — 直觀的、易於操作的
   - *This new AI tool is so intuitive, even beginners can use it.*
   - 這個新的 AI 工具非常直觀，連新手都能用。

## 📘 英文文法
### Used to vs. Be used to vs. Get used to
「Used to」表示過去的習慣或狀態；「be used to」表示習慣於某事；「「get used to」表示逐漸習慣某事的過程。
- I used to work long hours, but not anymore. (我以前常加班，但現在不會了。)
- She is used to debugging complex embedded systems. (她習慣除錯複雜的嵌入式系統。)
- It took me a while to get used to the new software interface. (我花了一點時間才習慣這個新軟體介面。)
### Conditional Sentences (Type 1 & 2)
Type 1 (If + 現在式, will + 原形動詞) 表可能發生的情況；Type 2 (If + 過去式, would + 原形動詞) 表與現在事實相反的假設。
- If we fix this bug, the system will run smoothly. (如果我們修好這個錯誤，系統就會順暢運行。)
- If I had more time, I would learn a new programming language. (如果我有更多時間，我就會學一種新的程式語言。)
- If the sensor detects movement, it will send an alert. (如果感測器偵測到移動，它就會發送警報。)

## 🗾 日文文法（N5）
### 動詞「ます」形與肯定/否定、過去式
**結構**：動詞(ます形) / 動詞(ません) / 動詞(ました) / 動詞(ませんでした)
「ます」表現在或未來肯定；「ません」表現在或未來否定；「ました」表過去肯定；「ませんでした」表過去否定。
- わたしは 毎日 コーヒーを 飲みます。 (我每天喝咖啡。)
- 昨日、会社へ 行きませんでした。 (我昨天沒去公司。)
### 名詞 + は + 名詞 + です / ではありません
**結構**：名詞1 + は + 名詞2 + です / ではありません
用來陳述「某人/某物是...」或「某人/某物不是...」的肯定或否定句型，是基本判斷句。
- わたしは エンジニアです。 (我是工程師。)
- これは わたしの パソコンではありません。 (這不是我的電腦。)
