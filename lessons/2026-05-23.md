## 📘 英文文法
### 條件句 (Conditional Type 1)
用於表達未來可能發生且結果真實的狀況。結構為 If + 主詞 + 現在簡單式, 主詞 + will + 原形動詞。
- If you finish the report, we will go for coffee. (如果你寫完報告，我們就去喝咖啡。)
- If the system crashes, I will restart it immediately. (如果系統當機，我會立刻重啟它。)
- If we optimize the code, it will run much faster. (如果我們優化程式碼，它會跑快很多。)
### 與「up」相關的動詞片語
許多動詞加上「up」形成片語，表達完成、增加或向上等意思，讓口語更生動。
- I need to catch up on my emails. (我需要處理完我的電子郵件。)
- Can you help me set up the new server? (你能幫我設定好新伺服器嗎？)
- Let's back up all the important files. (我們把所有重要檔案都備份起來吧。)

## 🗾 日文文法（N5）
### 「名詞 + が + あります / います」
**結構**：名詞 + が + あります / います
這個句型表示「有某物」或「有某人」。あります用於沒有生命的物體，います用於有生命的物體。
- ここにパソコンがあります。(這裡有電腦。)
- 会社にエンジニアがいます。(公司裡有工程師。)
### 「動詞て形 + ください」
**結構**：動詞て形 + ください
這個句型表示「請做某事」，用來禮貌地請求或指示對方執行某個動作。
- このファイルを保存してください。(請儲存這個檔案。)
- もう少し待ってください。(請再等一下。)
