# 2026-07-22（週三）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **implement** (v.) — 執行、實施（計畫或想法）
   - *We need to implement this new feature by Friday.*
   - 我們需要在週五前實施這個新功能。
2. **optimize** (v.) — 優化、最佳化（使效率更高）
   - *Can we optimize the code for better performance?*
   - 我們能優化程式碼來提升效能嗎？
3. **troubleshoot** (v.) — 排除故障、解決問題（技術層面）
   - *I spent all morning troubleshooting the network issue.*
   - 我整個早上都在排除網路問題。
4. **prototype** (n.) — 原型、樣本（初次設計或產品）
   - *The team is building a new IoT device prototype.*
   - 團隊正在製作一個新的物聯網裝置原型。
5. **collaborate** (v.) — 合作、協作（共同完成工作）
   - *We often collaborate with other teams on big projects.*
   - 我們常和其他團隊合作大型專案。
6. **feedback** (n.) — 反饋、意見（對某事的回應）
   - *Your feedback on the design is really helpful.*
   - 你對設計的反饋真的很有幫助。
7. **deadline** (n.) — 截止日期（必須完成工作的時間）
   - *I'm a bit stressed because the project deadline is tomorrow.*
   - 我有點壓力，因為專案截止日期是明天。
8. **innovative** (adj.) — 創新的、革新的（提出新方法或想法）
   - *We need more innovative solutions for renewable energy safety.*
   - 我們需要更創新的再生能源安全解決方案。
9. **efficient** (adj.) — 效率高的、有效率的（不浪費時間和精力）
   - *This new AI tool makes our workflow much more efficient.*
   - 這個新的 AI 工具讓我們的工作流程效率更高。
10. **monitor** (v.) — 監控、監視（持續觀察和檢查）
   - *We use sensors to monitor the temperature of the system.*
   - 我們使用感測器來監控系統的溫度。

## 📘 英文文法
### 被動語態 (Passive Voice)
當動作的執行者不重要或未知時，被動語態強調動作的承受者。
- The new system was developed by our R&D team. (新系統是由我們的研發團隊開發的。)
- User feedback is always valued by the product manager. (用戶反饋總是受到產品經理的重視。)
- Many IoT devices are powered by embedded systems. (許多物聯網裝置都是由嵌入式系統驅動的。)
### 情態動詞 (Modal Verbs) - must, should, can
情態動詞用於表達義務、建議、能力或可能性，讓語氣更自然。
- You must finish the report before leaving today. (你今天下班前必須完成報告。)
- We should check the code for potential bugs regularly. (我們應該定期檢查程式碼是否有潛在錯誤。)
- Can you help me set up this new development environment? (你能幫我設定這個新的開發環境嗎？)

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です / ではありません
**結構**：名詞１ + は + 名詞２ + です / ではありません
這個句型用來介紹或判斷事物，表示「名詞1是/不是名詞2」。
- 私はエンジニアです。 (我是工程師。)
- これは私のパソコンではありません。 (這不是我的電腦。)
### 動詞て形 + ください
**結構**：動詞て形 + ください
這個句型表示請求或指示對方做某事，語氣禮貌且常用。
- ちょっと待ってください。 (請稍等一下。)
- この資料を読んでください。 (請閱讀這份資料。)
