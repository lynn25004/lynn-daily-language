# 2026-05-30（週六）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **commute** (v./n.) — 上下班通勤（動詞/名詞）
   - *My daily commute takes about 30 minutes by train.*
   - 我每天搭火車通勤大約30分鐘。
2. **efficient** (adj.) — 高效的、有能力的
   - *We need an efficient way to manage project tasks.*
   - 我們需要一個有效率的方法來管理專案任務。
3. **troubleshoot** (v.) — 故障排除、解決問題
   - *Can you help me troubleshoot this network connection issue?*
   - 你能幫我排除這個網路連線問題嗎？
4. **integrate** (v.) — 整合、結合
   - *We need to integrate this new module into the existing system.*
   - 我們需要將這個新模組整合到現有系統中。
5. **prototype** (n.) — 原型、樣品
   - *The team is building a prototype for the new IoT device.*
   - 團隊正在為新的物聯網設備製作原型。
6. **seamless** (adj.) — 無縫的、流暢的
   - *We want a seamless user experience for our application.*
   - 我們希望我們的應用程式有流暢的使用者體驗。
7. **optimize** (v.) — 優化、使最佳化
   - *We need to optimize the code for better performance.*
   - 我們需要優化程式碼以獲得更好的效能。
8. **deadline** (n.) — 截止日期
   - *The project deadline is next Friday, so let's hurry!*
   - 專案截止日期是下週五，所以我們快點吧！
9. **feedback** (n.) — 反饋、意見
   - *Your feedback is crucial for improving our product.*
   - 你的反饋對改進我們的產品至關重要。
10. **device** (n.) — 裝置、設備
   - *This new smart device can monitor energy consumption.*
   - 這個新的智慧裝置可以監測能源消耗。

## 📘 英文文法
### Using "used to"
強調過去的習慣、常態或狀態，但現在已經不同或不再發生。
- I used to take the bus to work, but now I drive. (我以前搭公車上班，但現在開車。)
- This office building used to be a factory, right? (這棟辦公大樓以前是工廠，對吧？)
- He used to work on embedded systems a lot. (他以前常常在做嵌入式系統的工作。)
### Conditional Sentence Type 1
用來描述未來可能發生，且結果真實或很有可能發生的假設情況。
- If it rains tomorrow, I will work from home. (如果明天會下雨，我就在家工作。)
- If you finish the report, we can start the next phase. (如果你完成報告，我們就能開始下個階段。)
- If the IoT device connects, we will get data instantly. (如果物聯網設備連線，我們就會立刻收到數據。)

## 🇯🇵 日文單字（N5）
1. **おはようございます** / ohayou gozaimasu — 早上好
   - おはようございます！今日も一日がんばりましょう。
   - 早上好！今天也要努力一整天喔。
2. **かいしゃ** / 会社 / kaisha — 公司
   - わたしの会社はたいぺいにあります。
   - 我的公司在台北。
3. **しごと** / 仕事 / shigoto — 工作
   - きょうの仕事は午後5時に終わります。
   - 今天的工作會在下午5點結束。
4. **えき** / 駅 / eki — 車站
   - いえから駅まで歩いて5分です。
   - 從家裡走到車站需要5分鐘。
5. **でんしゃ** / 電車 / densha — 電車
   - 毎日電車で会社に行きます。
   - 我每天搭電車去公司。
6. **いそがしい** / 忙しい / isogashii — 忙碌的
   - こんしゅうはとても忙しいです。
   - 這週我非常忙碌。
7. **じかん** / 時間 / jikan — 時間
   - かいぎの時間は何時ですか？
   - 會議時間是幾點？
8. **はじめる** / 始める / hajimeru — 開始
   - 9時に仕事を始めます。
   - 我9點開始工作。
9. **おわる** / 終わる / owaru — 結束
   - 仕事が終わったら、れんらくします。
   - 工作結束後，我會聯絡你。
10. **べんきょう** / 勉強 / benkyou — 學習
   - 毎日日本語を勉強します。
   - 我每天學習日文。

## 🗾 日文文法（N5）
### N + は + N + です
**結構**：名詞 + は + 名詞 + です
用來介紹或說明事物，表示「A是B」或「A為B」的判斷句。
- これはわたしの会社です。 (這是我的公司。)
- わたしはエンジニアです。 (我是工程師。)
### Verb-ます form
**結構**：動詞(ます形)
動詞的禮貌形，表示現在或將來的肯定動作。也可用於習慣。
- 毎日日本語を勉強します。 (我每天學習日文。)
- 午後5時に仕事を終わります。 (我下午5點結束工作。)
