# 2026-07-03（週五）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **project** (n.) — 專案、計畫（5-12 字）
   - *My team handles a new IoT project.*
   - 我的團隊負責一個新的物聯網專案。
2. **schedule** (n.) — 時程表、計畫表
   - *Do we have a clear schedule for this?*
   - 我們有這個的明確時程表嗎？
3. **develop** (v.) — 開發、發展、研發
   - *We need to develop new software features.*
   - 我們需要開發新的軟體功能。
4. **solution** (n.) — 解決方案、解答
   - *We found a good solution to this problem.*
   - 我們找到了這個問題的好解決方案。
5. **feedback** (n.) — 回饋意見、反應
   - *I always welcome your honest feedback.*
   - 我隨時歡迎你誠實的回饋。
6. **integrate** (v.) — 整合、結合、併入
   - *Let's integrate this module smoothly.*
   - 讓我們順利整合這個模組吧。
7. **deadline** (n.) — 截止日期、最後期限
   - *The deadline is tight, but we can do it.*
   - 截止日期很趕，但我們可以做到。
8. **debug** (v.) — 偵錯、除錯（程式）
   - *I spent hours trying to debug this code.*
   - 我花了好幾個小時嘗試偵錯這段程式碼。
9. **collaborate** (v.) — 合作、協作、共同工作
   - *We should collaborate more on this task.*
   - 我們在這個任務上應該多合作。
10. **prototype** (n.) — 原型、樣品、雛形
   - *The team built the first prototype.*
   - 團隊完成了第一個原型。

## 📘 英文文法
### 動詞片語 (Phrasal Verbs)
動詞片語由動詞與介系詞或副詞組成，意思常與原動詞不同，讓表達更生動。
- Let's set up the new server. (我們來設定新伺服器吧。)
- Can you figure out this bug? (你能找出這個錯誤嗎？)
- I need to catch up on my emails. (我需要趕一下我的信件。)
### 表達義務或必要 (Expressing Obligation/Necessity)
使用 must, have to, need to 表達義務、必要或重要性，指出某事必須完成。
- We must finish this report today. (我們今天必須完成這份報告。)
- I have to attend a meeting now. (我現在必須去參加一個會議。)
- You need to update the software soon. (你很快就需要更新軟體。)

## 🇯🇵 日文單字（N5）
1. **かいしゃ** / 会社 / kaisha — 公司
   - 私は毎日会社に行きます。
   - 我每天去公司。
2. **しごと** / 仕事 / shigoto — 工作、事情
   - 今日の仕事は大変でした。
   - 今天的工作很辛苦。
3. **かいぎ** / 会議 / kaigi — 會議
   - 午後から会議があります。
   - 下午有會議。
4. **じかん** / 時間 / jikan — 時間
   - 仕事の時間が足りません。
   - 工作時間不夠。
5. **はじめる** / 始める / hajimeru — 開始（他動詞）
   - 新しいプロジェクトを始めます。
   - 開始新專案。
6. **おわる** / 終わる / owaru — 結束（自動詞）
   - 会議は５時に終わります。
   - 會議五點結束。
7. **きゅうけい** / 休憩 / kyuukei — 休息
   - 少し休憩しませんか。
   - 要不要休息一下？
8. **むずかしい** / 難しい / muzukashii — 困難的
   - この問題は少し難しいです。
   - 這個問題有點難。
9. **てつだう** / 手伝う / tetsudau — 幫忙、協助
   - 何か手伝いましょうか。
   - 需要我幫忙嗎？
10. **ほうこく** / 報告 / houkoku — 報告
   - 明日の朝、報告します。
   - 明天早上會報告。

## 🗾 日文文法（N5）
### 動詞て形 (動詞 Te 形)
**結構**：動詞て形
動詞て形有多種用途，例如連接動作、表示原因、提出請求或表示狀態。
- 資料を読んで、会議に行きます。(讀完資料再去開會。)
- コーヒーを飲んでください。(請喝咖啡。)
### 動詞ます形 (動詞 Masu 形)
**結構**：動詞ます形
動詞ます形是日文中最基本的禮貌形，用於日常對話中表達動作或狀態。
- 毎日、会社で働きます。(我每天在公司工作。)
- 新しい技術を勉強します。(我學習新的技術。)
