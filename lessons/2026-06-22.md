# 2026-06-22（週一）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **schedule** (n./v.) — 時間表；安排時間
   - *Let's check our team's schedule for this week.*
   - 我們來看看團隊這週的行程表吧。
2. **prioritize** (v.) — 優先處理；排序
   - *I need to prioritize tasks to meet the deadline.*
   - 我得把任務排好優先順序才能趕上截止日期。
3. **feedback** (n.) — 回饋；意見
   - *We'll get feedback on the new design soon.*
   - 我們很快會收到關於新設計的回饋。
4. **collaborate** (v.) — 合作；協作
   - *Engineers often collaborate on complex projects.*
   - 工程師們經常在複雜專案上合作。
5. **optimize** (v.) — 優化；最佳化
   - *We need to optimize the code for better performance.*
   - 我們需要優化程式碼來提升效能。
6. **implement** (v.) — 實施；執行
   - *It's time to implement the new safety protocol.*
   - 是時候實施新的安全協定囉。
7. **troubleshoot** (v.) — 排除故障；解決問題
   - *Can you help me troubleshoot this network issue?*
   - 你能幫我解決這個網路問題嗎？
8. **deadline** (n.) — 截止日期
   - *Our deadline for the IoT project is Friday.*
   - 我們這個 IoT 專案的截止日期是週五。
9. **brief** (v.) — 簡報；概述
   - *Let me brief you on the project status.*
   - 讓我向你簡要說明一下專案進度。
10. **sync up** (v.) — 同步；協調
   - *Let's sync up after lunch to discuss the details.*
   - 我們午餐後再同步一下細節吧。

## 📘 英文文法
### Used to (過去習慣)
表示過去的習慣、常態或狀態，但現在已不再是如此，語氣帶點懷舊。
- I used to code all night when I was younger. (我年輕的時候常常熬夜寫程式。)
- Our team used to have meetings every morning. (我們團隊以前每天早上都開會。)
- This company used to be smaller a few years ago. (這家公司幾年前規模小很多。)
### First Conditional (條件句 Type 1)
用於描述未來可能發生，且結果很可能實現的真實情況，結構為 If + 主詞 + 現在式, 主詞 + will/can/may + 原形動詞。
- If we finish the task early, we can relax a bit. (如果我們提早完成任務，就能稍微休息一下。)
- If the system crashes, I will restart it immediately. (如果系統當機，我會立刻重啟它。)
- If you have any questions, feel free to ask me anytime. (如果你有任何問題，隨時都可以問我喔。)

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作、任務
   - 私は毎日仕事をします。
   - 我每天都工作。
2. **かいぎ** / 会議 / kaigi — 會議
   - 今日は会議があります。
   - 今天有會議。
3. **じかん** / 時間 / jikan — 時間
   - 今、何時ですか。
   - 現在幾點了？
4. **つかう** / 使う / tsukau — 使用
   - このAIツールを使いますか。
   - 你會用這個AI工具嗎？
5. **わかる** / 分かる / wakaru — 知道、了解
   - 日本語が分かりますか。
   - 你懂日文嗎？
6. **よむ** / 読む / yomu — 閱讀
   - 私は技術書をよく読みます。
   - 我常讀技術書籍。
7. **かく** / 書く / kaku — 寫
   - レポートを書きます。
   - 我要寫報告。
8. **あたらしい** / 新しい / atarashii — 新的
   - 新しいプロジェクトが始まります。
   - 新專案要開始了。
9. **むずかしい** / 難しい / muzukashii — 困難的
   - この問題は少し難しいです。
   - 這個問題有點難。
10. **しゅうまつ** / 週末 / shuumatsu — 週末
   - 週末は何をしますか。
   - 週末你會做什麼？

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です
**結構**：名詞 + は + 名詞 + です
用於說明或定義一個事物，是日文中最基本的肯定句型，表達「A是B」的意思。
- これは私の仕事です。(這是我的工作。)
- あの人はエンジニアです。(那個人是工程師。)
### 動詞て形 + ください
**結構**：動詞て形 + ください
用於向對方提出請求或指示，表示「請做…」的意思，語氣客氣且常用於日常對話。
- ちょっと待ってください。(請稍等一下。)
- この資料を読んでください。(請閱讀這份資料。)
