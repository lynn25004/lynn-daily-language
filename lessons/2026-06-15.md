## 📘 英文文法
### 常用片語動詞（動詞 + up）
片語動詞由動詞加介副詞組成，意思常與字面不同，很常用於日常對話。
- Let's set up the new IoT device.（我們來設置新的物聯網設備。）
- Did you come up with a solution?（你有想出解決方案嗎？）
- Time to wrap up today's work.（是時候結束今天的工作了。）
### 未來式：going to vs. will
"going to" 表有計畫或已決定將來會做的事；"will" 則常用於即時決定、承諾或預測。
- I'm going to finish this report by noon.（我中午前會完成這份報告。）
- I think it will rain later.（我想晚點會下雨。）
- We are going to implement a new system next month.（我們下個月將實施新系統。）

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - きょうはしごとがあります。
   - 今天有工作。
2. **はじまる** / 始まる / hajimaru — 開始
   - かいぎはくじにはじまります。
   - 會議九點開始。
3. **おわる** / 終わる / owaru — 結束
   - しごとはなんじにおわりますか。
   - 工作幾點結束？
4. **つかう** / 使う / tsukau — 使用
   - このAIツールをつかいます。
   - 我會使用這個AI工具。
5. **むずかしい** / 難しい / muzukashii — 困難的、難的
   - このもんだいはすこしむずかしいです。
   - 這個問題有點難。
6. **かんたん** / 簡単 / kantan — 簡單的
   - それはかんたんだとおもいます。
   - 我覺得那很簡單。
7. **じかん** / 時間 / jikan — 時間
   - じかんがありますか。
   - 你有時間嗎？
8. **まいにち** / 毎日 / mainichi — 每天
   - まいにち にほんごをべんきょうします。
   - 我每天學習日語。
9. **すこし** / 少し / sukoshi — 一點點
   - すこしきゅうけいしましょう。
   - 我們稍微休息一下吧。
10. **だいじょうぶ** / 大丈夫 / daijoubu — 沒關係、沒問題
   - それはだいじょうぶです。
   - 那個沒關係。

## 🗾 日文文法（N5）
### 〜ます / 〜ません / 〜ました / 〜ませんでした (動詞肯定/否定/過去式)
**結構**：動詞(ます形)
動詞的「ます形」是基本禮貌體，用於表示現在、未來動作或習慣，其否定、過去式及過去否定式。
- わたしはまいにちコーヒーをのみます。（我每天喝咖啡。）
- きのう、しごとにいきませんでした。（昨天我沒去上班。）
### 〜ませんか (邀請/建議)
**結構**：動詞(ます形) + ませんか
用於禮貌地邀請對方一起做某事，或提出建議，語氣比「〜ましょう」更委婉。
- いっしょにランチをたべませんか。（要不要一起吃午餐？）
- すこしきゅうけいしませんか。（要不要稍微休息一下？）
