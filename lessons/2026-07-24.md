## 📘 英文文法
### 條件句 (Type 1)
用於表達未來可能發生且結果可預期的情況。句型為 If + 現在簡單式, 主詞 + will + 原形動詞。
- If you finish the report, we will go for coffee.（如果你完成報告，我們就去喝咖啡。）
- If the sensor works, the data will be accurate.（如果感測器能運作，資料就會是準確的。）
- If it rains tomorrow, I will work from home.（如果明天會下雨，我就在家工作。）
### 動詞片語：figure out & set up
figure out 指「弄清楚、理解」；set up 指「設置、建立」。兩者在工程師日常情境中很常用喔！
- I need to figure out why the system crashed.（我需要弄清楚系統為何當機。）
- Could you help me set up the new server?（你能幫我設定新的伺服器嗎？）
- Let's figure out a good time to meet.（我們來想個好時間見面吧。）

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作、職務
   - まい日、しごとをします。
   - 我每天都工作。
2. **かいしゃ** / 会社 / kaisha — 公司
   - わたしの かいしゃは ここです。
   - 我的公司在這裡。
3. **つかう** / 使う / tsukau — 使用、利用
   - この パソコンを つかいます。
   - 我會使用這台電腦。
4. **つくる** / 作る / tsukuru — 製作、創造
   - あたらしい ものを つくります。
   - 我製作新的東西。
5. **まなぶ** / 学ぶ / manabu — 學習、學會
   - わたしは にほんごを まなびたいです。
   - 我想學習日文。
6. **じかん** / 時間 / jikan — 時間
   - しごとの じかんは 8じです。
   - 工作時間是八點。
7. **まいにち** / 毎日 / mainichi — 每天
   - わたしは まいにち コーヒーを のみます。
   - 我每天都喝咖啡。
8. **あたらしい** / 新しい / atarashii — 新的、新鮮的
   - これは あたらしい プロジェクトです。
   - 這是一個新專案。
9. **だいじょうぶ** / 大丈夫 / daijoubu — 沒問題、沒關係
   - それ、だいじょうぶですか？
   - 那個，沒問題嗎？
10. **こまる** / 困る / komaru — 困擾、為難、煩惱
   - ちょっと こまっています。
   - 我有點困擾。

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です
**結構**：名詞A + は + 名詞B + です
這是日文最基本的肯定句型，用於說明事物的身份、名稱或性質。は是助詞，讀作「wa」。
- わたしは エンジニアです。（我是工程師。）
- これは パソコンです。（這是電腦。）
### 動詞(ます形) + たいです
**結構**：動詞ます形 + たいです
用於表達說話者想做某事的願望或意圖。把動詞的ます形後面的ます去掉，加上たいです。
- にほんごを べんきょうしたいです。（我想學習日文。）
- あたらしい プロジェクトを つくりたいです。（我想製作新專案。）
