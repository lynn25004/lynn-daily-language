## 📘 英文文法
### Conditional Sentences (Type 1)
用於描述未來可能發生的真實情境及其結果，句型為「If + 現在簡單式, will + 原形動詞」。
- If the code works, we will deploy it tomorrow. (如果程式碼有效，我們明天就會部署。)
- If you finish early, you can go home. (如果你提早完成，你可以回家。)
- If it rains, we'll stay indoors and play games. (如果下雨，我們就待在室內玩遊戲。)
### Phrasal Verbs with "up"
動詞與副詞"up"結合，常表示完成、建立、增加或達到某種狀態，語氣通常比較口語。
- Let's set up the new server this afternoon. (我們今天下午來架設新伺服器吧。)
- Can you come up with a new idea for the project? (你能為這個專案想出一個新點子嗎？)
- He wants to brush up his Japanese skills. (他想溫習一下他的日文能力。)

## 🗾 日文文法（N5）
### 動詞ます形（肯定、否定、過去式）
**結構**：動詞(ます形) + ます/ません/ました/ませんでした
用於表示禮貌的動詞現在肯定、現在否定、過去肯定、過去否定。是日常對話基礎。
- わたしはまいにちしごとをします。 (我每天工作。)
- きのうはかいぎがありませんでした。 (昨天沒有會議。)
### 名詞 + は + 名詞 + です/ではありません
**結構**：名詞１ + は + 名詞２ + です
「は」是主題標示助詞，用來表明句子的主題；「です」是禮貌的判斷助動詞，用於肯定句，「ではありません」用於否定句。
- わたしはエンジニアです。 (我是工程師。)
- これはわたしのノートではありません。 (這不是我的筆記本。)
