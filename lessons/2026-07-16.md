## 🇬🇧 英文單字（TOEIC 600-800）
1. **configure** (v.) — 配置、設定
   - *Hey, can you help me configure this new router?*
   - 嘿，你能幫我設定這台新路由器嗎？
2. **debug** (v.) — 除錯、偵錯
   - *I spent all morning trying to debug that tiny bug.*
   - 我花了整個早上試圖偵錯那個小錯誤。
3. **optimize** (v.) — 優化、最佳化
   - *We need to optimize the code for better performance.*
   - 我們需要優化程式碼以獲得更好的效能。
4. **interface** (n.) — 介面、接口
   - *The new user interface looks really clean and modern.*
   - 新的使用者介面看起來非常簡潔現代。
5. **sensor** (n.) — 感測器
   - *This IoT device has a temperature sensor built-in.*
   - 這個物聯網設備內建了一個溫度感測器。
6. **efficient** (adj.) — 高效率的、有效率的
   - *Working from home can be very efficient for some tasks.*
   - 居家辦公對某些任務來說可以非常有效率。
7. **prototype** (n.) — 原型、樣品
   - *We're building a prototype for the new smart device.*
   - 我們正在為新的智慧設備製作一個原型。
8. **deadline** (n.) — 截止日期
   - *The project deadline is next Friday, so let's hurry.*
   - 專案截止日期是下週五，所以我們快點吧。
9. **reliable** (adj.) — 可靠的、穩定的
   - *We need a reliable power source for our embedded system.*
   - 我們的嵌入式系統需要一個可靠的電源。
10. **collaborate** (v.) — 合作、協作
   - *Let's collaborate on this project to finish it faster.*
   - 讓我們在這個專案上合作，更快完成它。

## 📘 英文文法
### 現在完成式 (Present Perfect)
描述過去某個時間點開始，並持續到現在或與現在相關的動作或經驗。
- I have worked on several IoT projects.（我參與過幾個物聯網專案。）
- She has never used this AI tool before.（她以前從未使用過這個AI工具。）
- We've already optimized the system's performance.（我們已經優化了系統的效能。）
### 情態助動詞 (Modal Verbs: should, could, might)
表示建議、可能性或能力的助動詞，讓語氣更委婉，常用於日常交流。
- You should check the sensor data more often.（你應該更常檢查感測器數據。）
- We could try a different approach for debugging.（我們或許可以嘗試不同的除錯方法。）
- It might be a good idea to back up your files now.（現在備份你的檔案可能是個好主意。）

## 🗾 日文文法（N5）
### 動詞「〜ます」基本形
**結構**：動詞ます形 / 動詞ません形 / 動詞ました形 / 動詞ませんでした形
用於表示現在、未來或過去的動作，是禮貌體基本句型，讓對話更客氣。
- わたしは まいにち コーヒーを のみます。（我每天喝咖啡。）
- きのう、テレビを みませんでした。（昨天我沒看電視。）
### 「名詞 + を + 動詞」
**結構**：名詞 + を + 動詞
「を」是受格助詞，表示動詞的直接受詞，說明動作的對象或內容。
- わたしは パンを たべます。（我吃麵包。）
- 彼は しんぶんを よみます。（他讀報紙。）
