# 2026-05-21（週四）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **implement** (v.) — 執行、實施計畫
   - *We need to implement this new feature soon to improve performance.*
   - 我們需要盡快實施這個新功能以提升效能。
2. **optimize** (v.) — 讓事物變得更好
   - *Can we optimize this code for better speed and efficiency?*
   - 我們能優化這段程式碼以提高速度和效率嗎？
3. **troubleshoot** (v.) — 找出並解決問題
   - *I need to troubleshoot why the embedded system crashed suddenly.*
   - 我需要找出並解決為什麼嵌入式系統突然當機了。
4. **scenario** (n.) — 假設的情況或情境
   - *Let's consider a few different scenarios for the project's future.*
   - 讓我們考慮一下專案未來可能出現的幾種不同情境。
5. **integrate** (v.) — 將東西合併
   - *We need to integrate this new IoT module into the system.*
   - 我們需要將這個新的物聯網模組整合到系統中。
6. **efficient** (adj.) — 快速又好地完成
   - *This new process is very efficient and saves a lot of time.*
   - 這個新流程非常有效率，節省了很多時間。
7. **reliable** (adj.) — 可信賴、穩定
   - *Our server needs to be reliable for all renewable energy data.*
   - 我們的伺服器需要對所有再生能源數據保持可靠性。
8. **solution** (n.) — 問題的解決辦法
   - *I found a great solution for the bug in the software.*
   - 我為軟體中的這個錯誤找到了一個很棒的解決方案。
9. **feedback** (n.) — 意見或回報
   - *Please give me some immediate feedback on this prototype design.*
   - 請針對這個原型設計給我一些即時的回饋意見。
10. **update** (v./n.) — 更新資訊或版本
   - *Did you update the software on your embedded device yet?*
   - 你更新你嵌入式裝置上的軟體了嗎？

## 📘 英文文法
### Conditional Sentences (Type 1)
If + S + V (present), S + will + V (base)。描述未來可能發生的情況及其結果。
- If it rains tomorrow, we will stay home. (如果明天會下雨，我們就待在家。)
- If you finish early, you can leave. (如果你早點完成，就可以離開。)
- If the system works, it will save time. (如果系統能運作，它將節省時間。)
### Passive Voice
S + be + Vpp。強調動作的承受者，而非執行者，常用於客觀報告。
- The project was completed by the team. (這個專案由團隊完成。)
- New features are being developed this week. (新功能正在本週開發中。)
- The data is processed automatically. (數據會自動處理。)

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - わたしの しごとは エンジニアです。
   - 我的工作是工程師。
2. **かいしゃ** / 会社 / kaisha — 公司
   - わたしは かいしゃに いきます。
   - 我會去公司。
3. **でんき** / 電気 / denki — 電、電力
   - この きかいは でんきを つかいます。
   - 這台機器使用電力。
4. **つかう** / 使う / tsukau — 使用
   - パソコンを つかいます。
   - 使用電腦。
5. **べんり** / 便利 / benri — 方便
   - スマホは とても べんりです。
   - 智慧型手機很方便。
6. **あたらしい** / 新しい / atarashii — 新的
   - あたらしい パソコンが ほしいです。
   - 我想要一台新電腦。
7. **きのう** / 昨日 / kinou — 昨天
   - きのう、えいがを みました。
   - 昨天我看了電影。
8. **きょう** / 今日 / kyou — 今天
   - きょう、なにを しますか？
   - 今天要做什麼？
9. **あした** / 明日 / ashita — 明天
   - あした、かいぎが あります。
   - 明天有會議。
10. **べんきょう** / 勉強 / benkyou — 學習
   - にほんごを べんきょうします。
   - 我學習日文。

## 🗾 日文文法（N5）
### ～は～です (wa...desu)
**結構**：名詞1 + は + 名詞2 + です。
用於說明「A是B」，是最基本的肯定句型，介紹或描述事物。
- わたしは エンジニアです。(我是工程師。)
- これは わたしの パソコンです。(這是我的電腦。)
### ～を～ます (o...masu)
**結構**：名詞 + を + 動詞(ます形)。
表示動詞的直接受詞，指出動作的對象，常用於表達執行某動作。
- みずを のみます。(喝水。)
- ほんを よみます。(讀書。)
