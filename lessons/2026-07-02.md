## 🇬🇧 英文單字（TOEIC 600-800）
1. **integrate** (v.) — 整合、結合不同部分
   - *We need to integrate the new module.*
   - 我們需要整合這個新模組。
2. **prototype** (n.) — 原型、初步模型
   - *Let's build a quick prototype first.*
   - 我們先快速建一個原型吧。
3. **deploy** (v.) — 部署、上線
   - *When can we deploy this new feature?*
   - 我們什麼時候可以部署這個新功能？
4. **debug** (v.) — 除錯、找出錯誤
   - *I'm trying to debug this tricky bug.*
   - 我正在努力解決這個棘手的錯誤。
5. **optimize** (v.) — 優化、改進
   - *We need to optimize the system performance.*
   - 我們需要優化系統效能。
6. **feedback** (n.) — 反饋、意見回饋
   - *Your feedback is really helpful.*
   - 你的反饋真的很有幫助。
7. **solution** (n.) — 解決方案、答案
   - *Do you have any good solutions?*
   - 你有什麼好解決方案嗎？
8. **resource** (n.) — 資源、資料
   - *We need more learning resources.*
   - 我們需要更多學習資源。
9. **challenge** (n.) — 挑戰、難題

   - 這個專案是一個大挑戰。

   - *I hope to achieve my TOEIC goal!*
   - 我希望能達成我的多益目標！

## 📘 英文文法
### 情態助動詞 'Should' / 'Shouldn't'
'Should' 用於表達建議或義務，表示「應該」做某事；'Shouldn't' 則表示「不應該」。
- You should review your resume tonight. (你今晚應該複習一下你的履歷。)
- We should integrate AI tools into our workflow. (我們應該將AI工具整合到工作流程中。)
- You shouldn't worry too much about the interview. (你不應該太擔心面試。)
### 未來簡單式 'Will' / 'Be going to'
兩者都表示未來會發生的事情。'Will'常用於即時決定、預測；'Be going to'常用於已計畫、有根據的預測。
- I will start learning new embedded systems skills next week. (我下週將開始學習新的嵌入式系統技能。)
- Are you going to attend the IoT seminar tomorrow? (你明天會去參加物聯網研討會嗎？)
- I think the company will hire more engineers soon. (我想公司很快會僱用更多工程師。)

## 🇯🇵 日文單字（N5）
1. **べんきょう** / 勉強 / benkyou — 學習
   - まいにち にほんご を べんきょう します。
   - 我每天學習日文。
2. **わかる** / わかる / wakaru — 懂、明白
   - この もんだい が わかりません。
   - 我不懂這個問題。
3. **する** / する / suru — 做
   - なに を しますか。
   - 你要做什麼？
4. **むずかしい** / 難しい / muzukashii — 困難的
   - にほんご は むずかしい ですか。
   - 日文很難嗎？
5. **かんたん** / 簡単 / kantan — 簡單的
   - この しごと は かんたん です。
   - 這份工作很簡單。
6. **いま** / 今 / ima — 現在
   - いま、なんじ ですか。
   - 現在幾點了？
7. **しつもん** / 質問 / shitsumon — 問題、提問
   - しつもん が ありますか。
   - 你有問題嗎？
8. **おもしろい** / おもしろい / omoshiroi — 有趣的
   - AI ツール は おもしろい ですね。
   - AI 工具很有趣呢。
9. **しごと** / 仕事 / shigoto — 工作
   - あたらしい しごと を さがしています。
   - 我正在找新工作。
10. **かいしゃ** / 会社 / kaisha — 公司
   - どこ の かいしゃ ですか。
   - 是哪間公司？

## 🗾 日文文法（N5）
### 助詞「で」表示動作場所
**結構**：場所名詞 + で + 動詞(ます)
助詞「で」用來表示動作或事件發生的地點，例如在某個地方吃飯、學習或工作。
- かいしゃ で しごと を します。(在公司工作。)
- カフェ で コーヒー を のみます。(在咖啡廳喝咖啡。)
### 助詞「が」表示存在
**結構**：場所名詞 + に + もの/ひと + が + あります/います
助詞「が」搭配「あります」(無生命物體) 或「います」(有生命物體)，表示某物或某人存在於某地點。
- つくえ の うえ に 本 が あります。(桌子上有書。)
- へや に ねこ が います。(房間裡有貓。)
