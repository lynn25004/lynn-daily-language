# 2026-07-18（週六）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **configure** (v.) — 設定或調整設備
   - *I need to configure the new router for our office network.*
   - 我需要為我們辦公室的網路設定新的路由器。
2. **troubleshoot** (v.) — 找出並解決問題
   - *Can you help me troubleshoot this software bug?*
   - 你可以幫我解決這個軟體錯誤嗎？
3. **optimize** (v.) — 優化，使其達到最佳狀態
   - *We should optimize our code for better system performance.*
   - 我們應該優化我們的程式碼以獲得更好的系統性能。
4. **prototype** (n.) — 初步的模型或樣品
   - *The team built a prototype for the new IoT device.*
   - 團隊為新的物聯網設備建造了一個原型。
5. **renewable** (adj.) — 可再生的，永續的
   - *Solar power is a great example of renewable energy.*
   - 太陽能是再生能源的一個很好的例子。
6. **efficient** (adj.) — 效率高的，省時省力
   - *We need to find a more efficient way to manage data.*
   - 我們需要找到一個更有效率的數據管理方式。
7. **commute** (v.) — 通勤，上下班
   - *I usually commute by scooter; it saves time in traffic.*
   - 我通常騎機車通勤；這在交通上省了不少時間。
8. **hobby** (n.) — 閒暇時的嗜好或興趣
   - *What's your favorite hobby outside of work?*
   - 你工作之餘最喜歡的嗜好是什麼？
9. **deadline** (n.) — 截止日期，最終期限
   - *The project deadline is next Friday, so let's hurry up!*
   - 專案的截止日期是下週五，所以我們快點吧！
10. **sync** (v.) — 同步，使一致
   - *Did you sync your phone with the cloud backup service?*
   - 你把手機和雲端備份服務同步了嗎？

## 📘 英文文法
### 常用動詞片語：動詞 + up
許多動詞加上 'up' 會形成新的片語，常表示完成、建立、增加或結束。
- Let's set up the new development environment. (我們來設定新的開發環境吧。)
- Could you back up all your project files? (你可以備份你所有的專案檔案嗎？)
- I need to catch up on my emails after vacation. (我休假後需要趕上未讀的電子郵件。)
### 使用 'would' 表達假設或禮貌性請求
'Would' 可以用於禮貌地提出請求、邀請，或表達在假設情況下的行動。
- If I had more time, I would learn a new programming language. (如果我有更多時間，我會學習一種新的程式語言。)
- Would you mind helping me with this embedded system? (你介意幫我處理這個嵌入式系統嗎？)
- I would recommend checking the log files first. (我會建議你先檢查日誌檔。)

## 🗾 日文文法（N5）
### 「名詞１は名詞２です」
**結構**：名詞１ + は + 名詞２ + です
這是最基本的肯定句型，用來陳述「名詞１」是什麼或其身份。
- 私はエンジニアです。(我是工程師。)
- これは私のパソコンです。(這是我的電腦。)
### 「動詞(ます形) + ましょうか」
**結構**：動詞(ます形) + ましょうか
表示「我/我們來做某事好嗎？」用於主動提出幫忙或建議。
- 手伝いましょうか？ (我來幫忙好嗎？)
- コーヒーを入れましょうか？ (我來泡咖啡好嗎？)
