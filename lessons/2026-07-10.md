## 📘 英文文法
### 現在完成式 (Present Perfect)
用來表達過去發生的動作對現在造成的影響，或從過去持續到現在的經驗。
- I have worked on this project for months. (我已經在這個專案上工作好幾個月了。)
- She has just finished debugging the code. (她剛除錯完程式碼。)
- Have you ever used an AI tool for coding? (你曾經用過AI工具來寫程式嗎？)
### 動名詞 (Gerund) 作為主詞/受詞
將動詞加上-ing變成動名詞，可用來當作句子的主詞或受詞，表示「做某事」。
- Debugging takes a lot of time. (除錯需要很多時間。)
- I enjoy learning new programming languages. (我喜歡學習新的程式語言。)
- Developing embedded systems is my passion. (開發嵌入式系統是我的熱情。)

## 🗾 日文文法（N5）
### 〜ます / 〜ません / 〜ました / 〜ませんでした (Masu forms)
**結構**：動詞(ます形)
動詞「ます形」是用來表達現在或未來要做的事情，以及過去已做過的事情。
- わたしは まいにち にほんごを べんきょうします。 (我每天學習日文。)
- きのう、かいしゃへ いきました。 (昨天，我去了公司。)
### 名詞 + を + 動詞(ます)
**結構**：名詞 + を + 動詞(ます)
助詞「を」表示動詞的直接受詞，說明動詞動作的對象是什麼。
- ごはんを たべます。 (吃飯。)
- ホットコーヒーを のみますか。 (你喝熱咖啡嗎？)
