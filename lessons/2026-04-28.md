## 📘 英文文法
### Present Perfect (現在完成式)
表達經驗、從過去持續到現在的動作或影響，常用於描述已完成或未完成的事件。
- I have worked on IoT projects before. (我以前做過 IoT 專案。)
- She has updated the system. (她已經更新系統了。)
- We haven't fixed the bug yet. (我們還沒修好那個錯誤。)
### Modal Verbs: Can, Should, Must (情態助動詞：能、應該、必須)
用於表達能力、建議、義務或可能性，為主要動詞增添語氣與意義。
- You can use this tool to debug. (你可以用這個工具來除錯。)
- We should check the data security. (我們應該檢查資料安全。)
- Engineers must meet the deadline. (工程師必須在期限內完成。)

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です
**結構**：名詞 + は + 名詞 + です
用於說明主語是什麼，表達「A 是 B」的基本肯定句型，是日文最基礎的判斷句。
- これはIoTシステムです。(這是 IoT 系統。)
- わたしはエンジニアです。(我是工程師。)
### 名詞 + が + あります/います
**結構**：名詞 + が + あります/います
用於表達某物或某人存在於某地，あります用於非生物，います用於生物。
- へやにパソコンがあります。(房間裡有電腦。)
- かいぎしつにせんせいがいます。(會議室裡有老師。)
