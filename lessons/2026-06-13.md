# 2026-06-13（週六）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **Integrate** (v.) — 將不同部分結合在一起
   - *We need to integrate this new module into the system.*
   - 我們需要把這個新模組整合到系統裡。
2. **Optimize** (v.) — 讓性能或效率達到最佳
   - *I'm trying to optimize my code for better speed.*
   - 我正在嘗試優化我的程式碼以提升速度。
3. **Deploy** (v.) — 將系統或應用程式佈署
   - *We'll deploy the new software update next week.*
   - 我們下週會佈署新的軟體更新。
4. **Prototype** (n.) — 某個產品或系統的初步模型
   - *The team is building a new IoT prototype this weekend.*
   - 團隊這個週末正在製作一個新的物聯網原型。
5. **Troubleshoot** (v.) — 找出並解決問題或故障
   - *Can you help me troubleshoot this network connection issue?*
   - 你能幫我解決這個網路連線問題嗎？
6. **Weekend** (n.) — 一週的最後兩天（週六、週日）
   - *What are your plans for the upcoming long weekend?*
   - 你這個即將到來的長週末有什麼計畫？
7. **Relax** (v.) — 讓自己身心放鬆
   - *I just want to relax and read a book this Saturday.*
   - 我這個週六只想放鬆並讀本書。
8. **Project** (n.) — 需要努力和時間完成的任務
   - *I have a small side project related to smart home automation.*
   - 我有一個關於智慧家庭自動化的小型副專案。
9. **Explore** (v.) — 探索、探究或研究
   - *Let's explore some new AI tools for our daily tasks.*
   - 我們來探索一些日常任務用的新 AI 工具吧。
10. **Device** (n.) — 用於特定目的的機器或工具
   - *My smart home device stopped working suddenly.*
   - 我的智慧家庭設備突然停止運作了。

## 📘 英文文法
### 未來式 (will/be going to)
表達未來會發生的事情或計畫，'will' 更口語、更臨時，'be going to' 則用於既定計畫。
- I will probably work on my IoT project this afternoon. (我下午可能也會做我的物聯網專案。)
- We are going to visit the tech exhibition next month. (我們下個月要去參觀科技展。)
- He's going to troubleshoot the server issue later. (他待會要去排除伺服器問題。)
### 片語動詞 (Phrasal Verbs)
動詞搭配介系詞或副詞，意思常與單獨動詞不同，讓口語更自然。例如 'set up' (建立) 或 'figure out' (弄懂)。
- Can you help me set up this new embedded system? (你能幫我設定這個新的嵌入式系統嗎？)
- I need to figure out how to fix this bug. (我需要弄清楚如何修復這個錯誤。)
- Let's back up our data before we update the software. (我們在更新軟體前先備份資料吧。)

## 🗾 日文文法（N5）
### 動詞的ます形 (基本肯定/否定/過去)
**結構**：動詞ます / 動詞ません / 動詞ました / 動詞ませんでした
用於表達禮貌的肯定、否定、過去肯定與過去否定動作，是日文中最基本的動詞變化之一。
- コーヒーを飲みます。(我喝咖啡。)
- 本を読みませんでした。(我沒有讀書。)
### 助詞「を」與動詞
**結構**：名詞 + を + 動詞(ます形)
助詞「を」用來標示動詞的直接受詞，表示動詞作用的對象，是日文句型中非常重要的一環。
- 私は日本語を勉強します。(我學習日文。)
- 週末、映画を見ます。(週末我會看電影。)
