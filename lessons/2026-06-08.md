# 2026-06-08（週一）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **implement** (v.) — 執行、實施（計畫或功能）
   - *We need to implement this new feature by next week.*
   - 我們需要在下週前實施這個新功能。
2. **optimize** (v.) — 優化、最佳化（效能或效率）
   - *Can we optimize this code to run faster?*
   - 我們能優化這段程式碼讓它跑得更快嗎？
3. **troubleshoot** (v.) — 排除故障、解決問題（技術性）
   - *I spent all morning troubleshooting the server issues.*
   - 我整個早上都在排除伺服器問題。
4. **scenario** (n.) — 情境、設想（可能發生的狀況）
   - *Let's discuss a few different test scenarios for this project.*
   - 我們來討論幾個不同的專案測試情境吧。
5. **collaborate** (v.) — 合作、協作（共同完成任務）
   - *We should collaborate more often on these kinds of tasks.*
   - 我們應該更常在這類任務上合作。
6. **prototype** (n.) — 原型、樣本（產品的初步模型）
   - *The team is building a new IoT device prototype.*
   - 團隊正在製作一個新的物聯網設備原型。
7. **sustainable** (adj.) — 永續的、可持續的（對環境友善）
   - *We are looking for more sustainable energy solutions.*
   - 我們正在尋找更永續的能源解決方案。
8. **integrate** (v.) — 整合、結合（不同部分成一體）
   - *How can we integrate this module into the existing system?*
   - 我們如何將這個模組整合到現有系統中？
9. **feedback** (n.) — 回饋、意見（對工作或表現的評論）
   - *Please give me some feedback on my presentation draft.*
   - 請給我一些關於我的簡報草稿的回饋意見。
10. **deadline** (n.) — 截止日期（必須完成的期限）
   - *The deadline for this task is really tight, isn't it?*
   - 這項任務的截止日期真的很趕，不是嗎？

## 📘 英文文法
### 條件句 Type 1 (If + present, will + verb)
這種句型用來表達未來可能發生的事情及其結果。條件句使用現在簡單式，結果句則使用未來簡單式。
- If the code works, we'll celebrate tonight! (如果程式碼成功，我們今晚就慶祝！)
- If you finish early, we can grab some coffee. (如果你早點結束，我們可以去喝杯咖啡。)
- If it rains, the outdoor event will be cancelled. (如果下雨，戶外活動將會取消。)
### Used to / Be used to / Get used to
「Used to」表示過去的習慣或狀態，但現在已不再如此；「be used to」表示習慣於某事；「get used to」則表示逐漸習慣某事。
- I used to work late, but now I prefer to leave on time. (我以前常加班，但現在我喜歡準時下班。)
- I'm used to the long meetings now, so it's okay. (我現在已經習慣了冗長的會議，所以沒關係。)
- It took me a while to get used to the new system. (我花了一段時間才習慣新系統。)

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - きょうのしごとはたいへんです。
   - 今天的工作很辛苦。
2. **べんきょう** / 勉強 / benkyou — 學習
   - わたしはにほんごをべんきょうします。
   - 我學習日文。
3. **かいしゃ** / 会社 / kaisha — 公司
   - わたしの会社はあたらしいです。
   - 我的公司是新的。
4. **まいにち** / 毎日 / mainichi — 每天
   - まいにちコーヒーをのみます。
   - 我每天喝咖啡。
5. **いま** / 今 / ima — 現在
   - 今なんじですか。
   - 現在幾點了？
6. **じかん** / 時間 / jikan — 時間
   - 時間がありますか。
   - 有時間嗎？
7. **でんき** / 電気 / denki — 電、電燈
   - でんきをつけましょう。
   - 我們開燈吧。
8. **にほんご** / 日本語 / nihongo — 日文
   - 日本語はむずかしいです。
   - 日文很難。
9. **すき** / 好き / suki — 喜歡
   - わたしはラーメンが好きです。
   - 我喜歡拉麵。
10. **たいへん** / 大変 / taihen — 辛苦的、嚴重的
   - しごとは大変ですが、たのしいです。
   - 工作很辛苦，但很有趣。

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です
**結構**：名詞１ + は + 名詞２ + です
這個句型用來介紹或說明主語（名詞１）是什麼（名詞２），是非常基本且常用的肯定句型，表達「～是～」。
- わたしはエンジニアです。(我是工程師。)
- それはあたらしいシステムです。(那是新系統。)
### 動詞(ます形) + ましょう
**結構**：動詞(ます形) + ましょう
此句型表示提議或邀請對方「一起做某事」，或是勸誘對方「去做某事」，具有共同行動的語氣。
- しごとをしましょう。(我們工作吧。)
- いっしょにべんきょうしましょう。(我們一起學習吧。)
