## 📘 英文文法
### Phrasal Verbs with "up"
"up" 與動詞結合形成片語動詞，常表示完成、增加或向上。
- Let's set up the new server. (我們來設定新伺服器吧。)
- I need to look up this technical term. (我需要查這個技術詞彙。)
- Can you back up your files? (你能備份你的檔案嗎？)
### Using "should" for suggestions/advice
"should" 用來給予建議、提出義務或表達預期，語氣比 "must" 柔和。
- You should check the project requirements again. (你應該再檢查一次專案需求。)
- We should probably optimize this code. (我們大概應該優化這段程式碼。)
- They should be here by now, right? (他們現在應該已經到了吧？)

## 🗾 日文文法（N5）
### ~たいです (desire)
**結構**：動詞ます形 + たいです
表示說話者想要做某事，是一種直接且禮貌的表達願望的方式。
- 日本へ行きたいです。 (我想去日本。)
- 新しい技術を勉強したいです。 (我想學習新技術。)
### ~がわかります (understanding)
**結構**：名詞 + が + わかります
用來表達對某事物「理解」或「知道」，「が」是表示對象的助詞。
- 日本語がわかります。 (我懂日語。)
- このシステムがわかりますか。 (你懂這個系統嗎？)
