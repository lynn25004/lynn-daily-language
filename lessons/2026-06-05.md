# 2026-06-05（週五）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **optimize** (v.) — 使達到最佳狀態或效率
   - *We need to optimize our code for better performance.*
   - 我們需要優化程式碼來提升效能。
2. **integrate** (v.) — 整合、結合不同部分
   - *It's challenging to integrate new sensors into the system.*
   - 將新感測器整合到系統中很有挑戰性。
3. **troubleshoot** (v.) — 排除故障、找出並解決問題
   - *I spent hours trying to troubleshoot the network issue.*
   - 我花了數小時試圖排除網路問題。
4. **prototype** (n.) — 產品的初步模型或原型
   - *We're building a prototype for our new IoT device.*
   - 我們正在為新的物聯網設備製作原型。
5. **deadline** (n.) — 完成任務的最後期限
   - *The deadline for this project is next Friday, don't forget!*
   - 這個專案的截止日期是下週五，別忘了！
6. **collaborate** (v.) — 與他人一起合作完成任務
   - *We need to collaborate with the software team more closely.*
   - 我們需要與軟體團隊更緊密地合作。
7. **efficient** (adj.) — 高效的、節省時間和資源的
   - *His new design is very efficient and saves energy a lot.*
   - 他的新設計非常高效且節省大量能源。
8. **debug** (v.) — 找出並修正程式中的錯誤
   - *I have to debug my code before the demo tomorrow.*
   - 我必須在明天演示前除錯我的程式碼。
9. **flexible** (adj.) — 彈性的、能夠輕鬆適應的
   - *Our team offers flexible working hours, which is quite nice.*
   - 我們團隊提供彈性工時，這相當不錯。
10. **update** (v./n.) — 使某物變得最新、更新版本
   - *Always remember to update your software for better security and features.*
   - 永遠記得更新你的軟體以獲得更好的安全性和功能。

## 📘 英文文法
### 條件句 (第一型)
用於表達現在或未來可能發生的情況及其結果，結構是 "If + 主詞 + 現在簡單式, 主詞 + will + 原形動詞"。
- If it rains tomorrow, we will stay home. (如果明天下雨，我們就待在家。)
- If you finish your work early, we can grab a coffee. (如果你早點完成工作，我們可以去喝杯咖啡。)
- If the system crashes, I will call IT support immediately. (如果系統崩潰，我會立刻打給IT支援。)
### 動詞片語 "up" (常用)
動詞與介係詞"up"結合，形成新意思。常用於日常對話，詞義多變，例如設定、查詢、趕上等。
- Can you help me set up this new server? (你能幫我設定這台新伺服器嗎？)
- I need to look up some technical terms online. (我需要在網路上查一些技術術語。)
- Let's catch up over lunch next week. (我們下週午餐時聚聚吧。)

## 🗾 日文文法（N5）
### 動詞的禮貌形 (現在/過去)
**結構**：動詞ます形 / 動詞ません形 / 動詞ました形 / 動詞ませんでした形
用於表達禮貌的現在肯定、現在否定、過去肯定和過去否定動作，是最基本的動詞變化。
- 毎日日本語を勉強します。 (我每天學習日語。)
- 昨日会社に行きませんでした。 (我昨天沒有去公司。)
### 「～てください」 (請做...)
**結構**：動詞て形 + ください
用於禮貌地請求對方做某事，或提出指示。是日文裡很常見的請求句型，相當於中文的「請～」。
- ここに座ってください。 (請坐這裡。)
- 資料を読んでください。 (請讀資料。)
