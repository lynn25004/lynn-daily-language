# 2026-06-04（週四）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **optimize** (v.) — 優化、改善系統或流程
   - *I need to optimize this code for better performance and speed.*
   - 我需要優化這段程式碼以提升效能和速度。
2. **implement** (v.) — 實施、執行計畫或策略
   - *We plan to implement the new feature next week. It's exciting!*
   - 我們計畫下週實施這項新功能。真令人興奮！
3. **troubleshoot** (v.) — 故障排除、解決問題
   - *Can you help me troubleshoot this network issue? It's quite slow.*
   - 你能幫我排除這個網路問題嗎？它很慢。
4. **renewable** (adj.) — 可再生的、永續的
   - *More companies are investing in renewable energy sources like solar.*
   - 越來越多公司投入太陽能等再生能源。
5. **prototype** (n.) — 原型、初步模型
   - *We're building a prototype for the new IoT device. Wish us luck!*
   - 我們正在為新的 IoT 設備製作一個原型。祝我們好運！
6. **collaborate** (v.) — 合作、協同工作
   - *Let's collaborate on this project to finish it faster together.*
   - 我們來合作這個專案，一起更快完成吧。
7. **deadline** (n.) — 截止日期、最後期限
   - *I'm trying to meet a tight deadline for my report. Almost there!*
   - 我正努力趕上報告的緊迫期限。快完成了！
8. **innovative** (adj.) — 創新的、有創意的
   - *His ideas are always very innovative and inspiring to everyone.*
   - 他的想法總是很有創意且鼓舞人心。
9. **efficiency** (n.) — 效率、效能
   - *We should improve the efficiency of our daily workflow. Any ideas?*
   - 我們應該提升日常工作流程的效率。有什麼想法嗎？
10. **hobby** (n.) — 嗜好、興趣
   - *What's your favorite hobby outside of work? Mine is reading.*
   - 工作之餘你最喜歡的嗜好是什麼？我的是閱讀。

## 📘 英文文法
### 過去習慣 (used to)
表達過去常做但現在不再發生的習慣或狀態，或過去真實但現在已非的事實。
- I used to play video games all night when I was younger. (我年輕時常熬夜玩電動。)
- This building used to be a small factory, but now it's a cafe. (這棟建築以前是間小工廠，但現在是咖啡廳。)
- Engineers used to rely more on manual testing before automation. (工程師以前在自動化之前更依賴手動測試。)
### 兩者擇一 (either... or...)
用來連接兩個選項，表示「不是A就是B」，強調在兩個可能性中選擇一個。
- You can either call me or send me an email to confirm. (你可以打電話給我或寄電子郵件給我確認。)
- We can either fix the bug now or postpone it until tomorrow. (我們現在可以修復這個錯誤，不然就延後到明天。)
- Either the system is down or my internet connection is slow. (不是系統當機，就是我的網路連線很慢。)

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - 毎日、仕事が楽しいです。
   - 每天工作都很開心。
2. **やすみ** / 休み / yasumi — 休息、假日
   - 明日は休みだから、ゆっくりします。
   - 明天是假日，所以我要好好休息。
3. **べんきょう** / 勉強 / benkyou — 學習
   - 日本語の勉強は面白いです。
   - 學習日文很有趣。
4. **じかん** / 時間 / jikan — 時間
   - どのくらい時間がかかりますか。
   - 大概要花多少時間呢？
5. **たのしい** / 楽しい / tanoshii — 快樂的、愉快的
   - 友達と話すのは楽しいです。
   - 和朋友聊天很開心。
6. **よむ** / 読む / yomu — 閱讀
   - 私は本を読むのが好きです。
   - 我喜歡讀書。
7. **かく** / 書く / kaku — 寫
   - 毎日、日本語を少し書きます。

8. **いま** / 今 / ima — 現在

   - 先生の話がよく分かります。
   - 我很懂老師說的話。

## 🗾 日文文法（N5）
### 存在句 (N + が + あります/います)
**結構**：名詞 + が + あります (非生物) / います (生物)
表示某處有「什麼東西」或「誰」，用來表達事物的存在。
- 私の部屋に本がたくさんあります。 (我的房間裡有很多書。)
- 会社にエンジニアが三人います。 (公司裡有三位工程師。)
### 希望句 (Vます + たいです)
**結構**：動詞ます形 + たいです
表示說話者「想做」某個動作或事情，表達願望或慾望。
- 週末に映画を見たいです。 (週末我想看電影。)
- もっと日本語を勉強したいです。 (我想學更多日文。)
