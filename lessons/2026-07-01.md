# 2026-07-01（週三）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **optimize** (v.) — 讓…最佳化，達到最好
   - *Let's optimize this code to run faster and smoother.*
   - 我們來最佳化這段程式碼，讓它跑得更快更順暢。
2. **debug** (v.) — 除錯，找出並修復錯誤
   - *I need to debug this circuit board after lunch.*
   - 我午飯後需要為這塊電路板除錯。
3. **deadline** (n.) — 截止日期，專案結束時間
   - *The project deadline is Friday, so let's keep going!*
   - 專案截止日是星期五，所以我們繼續努力吧！
4. **collaborate** (v.) — 合作，共同完成某事
   - *We should collaborate more often on these challenging tasks.*
   - 對於這些有挑戰性的任務，我們應該多合作。
5. **innovative** (adj.) — 創新的，有新意的
   - *That's a truly innovative solution for our IoT device.*
   - 那是我們物聯網設備一個真正創新的解決方案。
6. **feature** (n.) — 功能，產品的特色
   - *This new feature will make the app super useful.*
   - 這個新功能會讓這個應用程式超級實用。
7. **feedback** (n.) — 回饋，意見或建議
   - *Can you give me some feedback on my presentation?*
   - 你能給我一些關於我簡報的回饋嗎？
8. **deploy** (v.) — 部署，佈署軟體或系統
   - *We're ready to deploy the new software update.*
   - 我們準備好部署新的軟體更新了。
9. **efficient** (adj.) — 高效的，節省時間或資源
   - *Using AI tools can make our workflow very efficient.*
   - 使用 AI 工具可以讓我們的工作流程非常高效。
10. **prototype** (n.) — 原型，初步模型或版本
   - *Let's build a prototype to test the concept first.*
   - 我們不如先建一個原型來測試這個概念。

## 📘 英文文法
### Used to (過去習慣)
表示過去常做的事或過去的狀態，但現在已不再如此。語氣輕鬆，用於日常。
- I used to work late every night, but not anymore. (我以前常加班到很晚，但現在不會了。)
- This old factory used to be a famous bakery. (這間老工廠以前是間有名的麵包店。)
- Did you use to play video games all day? (你以前是不是整天都在打電動？)
### Modal Verbs for Suggestions/Advice (should/could)
Should 表示「應該」，提出建議或義務；Could 表示「可以」，提出可能性或禮貌的建議。
- You should take a break, you look tired. (你應該休息一下，你看起來很累。)
- We could try a different approach to solve this bug. (我們可以嘗試不同的方法來解決這個錯誤。)
- What should I do to improve my English? (我該怎麼做才能增進我的英文？)

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - 毎日、仕事があります。
   - 每天都有工作。
2. **じかん** / 時間 / jikan — 時間
   - 今、何時ですか？
   - 現在幾點了？
3. **かいぎ** / 会議 / kaigi — 會議
   - 午後に会議があります。
   - 下午有會議。
4. **きゅうけい** / 休憩 / kyuukei — 休息
   - ちょっと休憩しましょう。
   - 我們稍微休息一下吧。
5. **パソコン** / パソコン / pasokon — 電腦 (PC)
   - 私のパソコンは新しいです。
   - 我的電腦是新的。
6. **あたらしい** / 新しい / atarashii — 新的
   - 新しいプロジェクトが始まります。
   - 新專案要開始了。
7. **べんり** / 便利 / benri — 方便的
   - このツールはとても便利です。
   - 這個工具很方便。
8. **がんばる** / 頑張る / ganbaru — 努力，加油
   - 仕事、頑張ります！
   - 工作，我會努力的！
9. **だいじょうぶ** / 大丈夫 / daijoubu — 沒關係，不要緊
   - 大丈夫ですか？
   - 你還好嗎？
10. **まなぶ** / 学ぶ / manabu — 學習
   - 毎日、日本語を学びます。
   - 每天學習日文。

## 🗾 日文文法（N5）
### 〜は〜です (主題句型)
**結構**：名詞１ + は + 名詞２ + です
用來介紹或說明事物，表示「名詞１是名詞２」。這是最基礎的句型，非常常用。
- 私はエンジニアです。(我是工程師。)
- これは私のパソコンです。(這是我的電腦。)
### 名詞 + を + 動詞(ます形) (受詞句型)
**結構**：名詞 + を + 動詞(ます形)
助詞「を」表示它前面的名詞是動詞的受詞，說明動作的對象。是日文動詞句型的基礎。
- コーヒーを飲みます。(我喝咖啡。)
- 日本語を勉強します。(我學習日文。)
