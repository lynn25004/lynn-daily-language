## 📘 英文文法
### 條件句 Type 1 (未來可能發生的情況)
用於描述未來可能發生的事情及其結果，強調實際可能性，結構為 If + 現在簡單式, will + 原形動詞。
- If it rains tomorrow, we will work from home. (如果明天會下雨，我們就在家工作。)
- If you finish the report, you can take a break. (如果你寫完報告，就可以休息了。)
- If the system crashes, we will restart it immediately. (如果系統當機，我們就會立刻重啟。)
### 常用片語動詞 (Phrasal Verbs)
動詞與介系詞或副詞結合，產生新的意義，常用於口語和日常交流，讓表達更自然。
- Let's set up a meeting for next Tuesday. (我們把會議訂在下週二吧。)
- I need to figure out how to fix this bug. (我需要想辦法解決這個錯誤。)
- Don't give up on your dreams, okay? (別放棄你的夢想，好嗎？)

## 🗾 日文文法（N5）
### N1 は N2 です
**結構**：名詞1 + は + 名詞2 + です
這個句型用來介紹或描述事物，表示「名詞1是名詞2」，是日語最基礎的判斷句。
- わたしは エンジニア です。(我是工程師。)
- これは わたしの パソコン です。(這是我的電腦。)
### 動詞て形 + ください
**結構**：動詞て形 + ください
這個句型用來禮貌地請求或指示對方做某事，表示「請做...」。て形是動詞的一種變化形式。
- ちょっと まって ください。(請稍等一下。)
- このほんを よんで ください。(請讀這本書。)
