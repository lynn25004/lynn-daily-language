# 2026-06-25（週四）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **implement** (v.) — 執行、實施
   - *We need to implement this new feature by next week.*
   - 我們下週前要實作這個新功能。
2. **optimize** (v.) — 優化、最佳化
   - *Can you help me optimize this code for better performance?*
   - 你能幫我優化這段程式碼以提升效能嗎？
3. **troubleshoot** (v.) — 排除故障
   - *I spent hours trying to troubleshoot the network issue.*
   - 我花了數小時試著排除網路問題。
4. **prototype** (n.) — 原型、雛形
   - *We're building a prototype for our new IoT device.*
   - 我們正在為新的物聯網裝置打造原型。
5. **integrate** (v.) — 整合、集成
   - *We plan to integrate AI into our renewable energy system.*
   - 我們計劃將AI整合到我們的再生能源系統中。
6. **feature** (n.) — 特色、功能
   - *This new smartphone has some really cool AI features.*
   - 這款新智慧型手機有一些非常酷的AI功能。
7. **commute** (n./v.) — 通勤
   - *My daily commute takes about an hour, mostly by train.*
   - 我每天通勤大約一小時，主要搭火車。
8. **deadline** (n.) — 截止日期
   - *The project deadline is approaching fast, we must hurry!*
   - 專案截止日期快到了，我們必須加快速度！
9. **efficient** (adj.) — 高效率的
   - *We need an efficient way to manage data from sensors.*
   - 我們需要有效率的方式來管理感測器數據。
10. **monitor** (v.) — 監控、監視
   - *We use sensors to monitor the air quality in the office.*
   - 我們用感測器監控辦公室的空氣品質。

## 📘 英文文法
### 過去完成式 (Past Perfect)
用於描述在過去某個時間點之前，已經完成或發生的動作。
- By the time I arrived, they had already fixed the bug. (我到的時候，他們已經修好那個錯誤了。)
- She had finished her report before the deadline. (她在截止日期前就完成了報告。)
- We had never seen such a complex system until then. (在那之前，我們從未見過如此複雜的系統。)
### 動名詞 (Gerund) 作為主詞或受詞
動名詞是動詞的-ing形式，可當名詞使用，表示一個動作或活動。
- Debugging can be really time-consuming sometimes. (除錯有時候真的會很耗時。)
- I enjoy learning about new IoT devices. (我很喜歡學習新的物聯網裝置。)
- Designing embedded systems is my passion. (設計嵌入式系統是我的熱情所在。)

## 🇯🇵 日文單字（N5）
1. **べんきょう** / 勉強 / benkyou — 學習
   - 毎日、日本語をべんきょうします。
   - 我每天學習日文。
2. **しごと** / 仕事 / shigoto — 工作
   - エンジニアのしごとはとてもおもしろいです。
   - 工程師的工作很有趣。
3. **まいにち** / 毎日 / mainichi — 每天
   - わたしはまいにちコーヒーをのみます。
   - 我每天喝咖啡。
4. **じかん** / 時間 / jikan — 時間
   - じかんがありますか。
   - 你有時間嗎？
5. **べんり** / 便利 / benri — 方便
   - このスマホはとてもべんりです。
   - 這支智慧型手機非常方便。
6. **でんき** / 電気 / denki — 電、電燈
   - でんきをつけてください。
   - 請開燈。
7. **つくる** / 作る / tsukuru — 製作、製造
   - あたらしいロボットをつくりたいです。
   - 我想製作新的機器人。
8. **つかう** / 使う / tsukau — 使用
   - このツールをつかいましょう。
   - 我們來使用這個工具吧。
9. **かいしゃ** / 会社 / kaisha — 公司
   - わたしのかいしゃはたいぺいにあります。
   - 我的公司在台北。
10. **たいへん** / 大変 / taihen — 辛苦、很... (程度副詞)
   - 日本語のぶんぽうはたいへんですね。
   - 日文文法很難/很辛苦呢。

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です
**結構**：名詞 + は + 名詞 + です
這是日文最基本的肯定句型，用於表示「A是B」或介紹事物。
- わたしはエンジニアです。(我是工程師。)
- これはわたしのノートパソコンです。(這是我的筆記型電腦。)
### 動詞て形 + ください
**結構**：動詞て形 + ください
將動詞變成て形後，加上「ください」表示禮貌的請求或指示，意思是「請做...」。
- ここにすわってください。(請坐這裡。)
- ゆっくりはなしてください。(請慢慢說。)
