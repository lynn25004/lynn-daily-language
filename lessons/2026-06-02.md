# 2026-06-02（週二）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **efficient** (adj.) — 效率高的
   - *This new programming language allows us to write more efficient and cleaner code.*
   - 這款新的程式語言能讓我們寫出更有效率、更簡潔的程式碼。
2. **optimize** (v.) — 優化、最佳化
   - *We need to optimize our backend system to handle increased user traffic more smoothly.*
   - 我們需要優化後端系統，以更流暢地處理增加的使用者流量。
3. **prototype** (n.) — 原型、樣本
   - *The team will present the first working prototype of our smart home device next week.*
   - 團隊下週將展示我們智慧家庭裝置的第一個運作原型。
4. **debug** (v.) — 除錯、偵錯
   - *I spent all morning trying to debug that elusive bug in the embedded system.*
   - 我整個早上都在嘗試除錯嵌入式系統中那個難搞的錯誤。
5. **renewable** (adj.) — 可再生的
   - *Many companies are investing heavily in renewable energy sources like solar power.*
   - 許多公司現在正大量投資太陽能等再生能源。
6. **curious** (adj.) — 好奇的
   - *I'm really curious about how those new AI tools can improve our workflow.*
   - 我對那些新的 AI 工具如何改進我們的工作流程感到非常好奇。
7. **commute** (n.) — 通勤、上下班路程
   - *My morning commute usually takes about thirty minutes, giving me time for podcasts.*
   - 我早上的通勤時間通常約三十分鐘，讓我有時間聽播客。
8. **deadline** (n.) — 截止日期
   - *We have a really tight deadline for the upcoming IoT project, so let's work fast.*
   - 我們這個即將到來的 IoT 專案截止日期很緊，所以我們要加快速度。
9. **collaborate** (v.) — 合作、協作
   - *Let's collaborate closely with the marketing team to meet customer needs.*
   - 讓我們與行銷團隊密切合作，以滿足客戶需求。
10. **update** (v.) — 更新
   - *Did you remember to update your operating system and all your development tools this morning?*
   - 你今天早上記得更新你的作業系統和所有開發工具了嗎？

## 📘 英文文法
### Conditional Sentences Type 1 (If + present simple, will + base verb)
用來表達在未來有可能發生且會產生特定結果的事情。
- If it rains tomorrow, I will work from home. (如果明天會下雨，我就在家工作。)
- If you finish early, we can grab coffee. (如果你早點完成，我們可以去喝咖啡。)
- If the code works, we will save time. (如果程式碼有效，我們會省下時間。)
### Phrasal Verbs: set up & figure out
動詞與介係詞或副詞組合成新意義，常用於日常對話中。
- Can you help me set up this new device? (你能幫我設定這台新裝置嗎？)
- I need to figure out how to fix this bug. (我需要弄清楚如何修復這個錯誤。)
- Let's set up a meeting for next Tuesday. (我們來安排下週二的會議吧。)

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です
**結構**：名詞１ + は + 名詞２ + です
用來介紹或說明主語（名詞１）是什麼（名詞２），是日文最基本的肯定句型。
- これはペンです。 (這是筆。)
- 私はエンジニアです。 (我是工程師。)
### 動詞-ます形 (ません)
**結構**：動詞-ます形 + ません
動詞「ます形」的否定形式，表示不進行某動作或習慣，用於現在或將來。
- コーヒーを飲みません。 (我不喝咖啡。)
- 明日、仕事をしません。 (我明天不工作。)
