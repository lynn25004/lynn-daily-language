## 📘 英文文法
### 條件句 (Conditional Type 1)
用於表達未來可能發生、條件成立時的結果。通常結構為「If + 現在簡單式, 主詞 + will/can/may + 原形動詞」。
- If I finish my work early, I will go for a run. (如果我早點完成工作，我就去跑步。)
- If you need help with the code, I will be happy to assist. (如果你需要程式碼的幫助，我很樂意協助。)
- If we optimize the system, it will save a lot of energy. (如果我們優化這個系統，它會節省很多能源。)
### 動詞片語: look forward to
表示非常期待某事發生。後面接名詞或動名詞 (V-ing)，表示對未來事件的興奮或期盼。
- I'm really looking forward to the weekend! (我真的很期待週末！)
- I look forward to discussing your project ideas. (我期待討論你的專案想法。)
- We are looking forward to the new AI tool release. (我們期待新 AI 工具的發布。)

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です
**結構**：名詞１ + は + 名詞２ + です
這是最基礎的句型，用於說明或介紹，表示「名詞1是名詞2」。は讀作「wa」。
- 私はエンジニアです。（我是工程師。）
- これは日本語の本です。（這是日文的書。）
### 動詞(ます形) + ましょう
**結構**：動詞ます形 + ましょう
表達「一起做...吧」的提議或建議，或是自己決定「做...吧」。用於邀請或提出行動。
- 一緒に日本語を勉強しましょう。（我們一起學日文吧。）
- 週末、新しいIoTプロジェクトを始めましょう。（週末，我們開始新的IoT專案吧。）
