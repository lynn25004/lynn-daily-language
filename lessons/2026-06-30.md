# 2026-06-30（週二）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **prototype** (n.) — 原型、雛形（產品初期版本）
   - *We're building a prototype for the new IoT device.*
   - 我們正在為新的物聯網設備製作原型。
2. **debug** (v.) — 除錯、偵錯（軟體問題）
   - *I spent hours trying to debug this code yesterday.*
   - 我昨天花了數小時試圖調試這段程式碼。
3. **implement** (v.) — 實施、執行（計畫或系統）
   - *We need to implement the new security features soon.*
   - 我們需要盡快實施新的安全功能。
4. **scenario** (n.) — 情境、設想（可能發生的情況）
   - *Let's consider a few different user scenarios.*
   - 讓我們考慮幾種不同的用戶情境。
5. **optimize** (v.) — 優化、最佳化（提高效率）
   - *We should optimize the system for better performance.*
   - 我們應該優化系統以獲得更好的性能。
6. **reliable** (adj.) — 可靠的、穩定的（值得信賴）
   - *This power supply is really reliable for my project.*
   - 這個電源對我的專案來說非常可靠。
7. **commute** (n.) — 通勤（上下班的路程）
   - *My morning commute is usually about 30 minutes.*
   - 我早上的通勤時間通常約 30 分鐘。
8. **catch up** (v. phr.) — 敘舊、了解最新情況
   - *Let's catch up over coffee this weekend!*
   - 這個週末我們喝咖啡聊聊近況吧！
9. **hobby** (n.) — 嗜好、興趣（閒暇活動）
   - *What's your favorite hobby outside of work?*
   - 你工作之餘最喜歡的嗜好是什麼？
10. **update** (v./n.) — 更新、最新消息
   - *Do you have any updates on the project status?*
   - 關於專案進度，你有任何最新消息嗎？

## 📘 英文文法
### 未來計畫 (be going to / Present Continuous for future)
be going to 用於表達預定或有根據的未來計畫，Present Continuous 可用於表達已安排好的未來行程。
- I'm going to work on my AI side project tonight. (我今晚要弄我的 AI 副專案。)
- We are meeting with the client next Monday. (我們下週一要和客戶見面。)
- What are you going to do this coming weekend? (你這個週末打算做什麼？)
### 條件句 (Type 1 - Real Conditionals)
用於描述真實或極可能發生的情況及其結果，結構為「If + 現在簡單式, 主詞 + will + 原形動詞」。
- If the system works, we will launch it next month. (如果系統運作正常，我們下個月就會發布。)
- If you have time, can you help me debug this code? (如果你有時間，可以幫我調試這段程式碼嗎？)
- If I finish early, I'll go for a run. (如果我早點完成，我就去跑步。)

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - まいにち しごと が あります。
   - 我每天都有工作。
2. **しゅうまつ** / 週末 / shuumatsu — 週末
   - しゅうまつ は なに を しますか。
   - 週末你要做什麼？
3. **たのしい** / 楽しい / tanoshii — 快樂的、有趣的
   - この べんきょう は とても たのしい です。

   - いま なんじかん ですか。

## 🗾 日文文法（N5）
### 動詞的「ます」形和「ません」形
**結構**：動詞ます形 / 動詞ません形
「ます」用於肯定且禮貌的語氣，表示現在或未來的動作。「ません」則用於否定且禮貌的語氣。
- わたしは 毎日 コーヒーを 飲みます。(我每天喝咖啡。)
- わたしは 日本語を 勉強しません。(我不學習日語。)
### ~たいです (想做某事)
**結構**：動詞ます形去ます + たいです
這個句型用於表達說話者想做某事的願望或希望，是 N5 階段非常實用的表達方式。
- わたしは 新しい スマホを 買いたいです。(我想要買新手機。)
- 週末、映画を 見たいです。(週末我想看電影。)
