## 📘 英文文法
### Conditional Sentences Type 1 (If + Present Simple, Will + Verb)
用於描述未來可能發生的真實情況，若某條件達成，則某結果會發生。
- If you finish early, we can grab coffee. (如果你早點完成，我們可以去喝咖啡。)
- If the system crashes, I will restart it immediately. (如果系統當機，我會立刻重啟它。)
- If we collaborate more, the project will be smoother. (如果我們多合作，專案會更順利。)
### Phrasal Verbs with "up" (e.g., set up, come up with, speed up)
由動詞和介詞或副詞組成，意思常與原動詞不同，讓表達更自然。
- Let's set up the new server this afternoon. (我們今天下午來設定新伺服器吧。)
- Can you come up with a solution for this bug? (你能想出這個錯誤的解決方案嗎？)
- We need to speed up the development process. (我們需要加快開發進度。)

## 🗾 日文文法（N5）
### 動詞(ます形) + たいです
**結構**：動詞(ます形) + たいです
當你想要表達自己想做某事的願望時使用，輕鬆表達你的意願。
- 日本語を勉強したいです。 (我想學習日語。)
- 新しいAIツールを使いたいです。 (我想使用新的AI工具。)
### 名詞 + が あります/います
**結構**：名詞 + が あります / います
「あります」用於非生物，「います」用於生物，表示存在或擁有某物/某人。
- 私の部屋にパソコンがあります。 (我的房間有電腦。)
- 会社に新しいエンジニアがいます。 (公司有一位新工程師。)
