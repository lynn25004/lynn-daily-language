# 2026-05-25（週一）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **implement** (v.) — 執行、實施
   - *We need to implement this new feature by next week.*
   - 我們下週前要實施這個新功能。
2. **optimize** (v.) — 優化、最佳化
   - *Can you help me optimize this code for better performance?*
   - 你能幫我優化這段程式碼以提升效能嗎？
3. **troubleshoot** (v.) — 排除故障
   - *I spent hours trying to troubleshoot that network issue.*
   - 我花了好幾個小時試圖排除那個網路問題。
4. **deadline** (n.) — 截止日期
   - *The project deadline is approaching fast, we need to hurry.*
   - 專案截止日期快到了，我們得加快速度。
5. **collaborate** (v.) — 合作、協作
   - *It's great to collaborate with such a supportive team.*
   - 與這麼棒的團隊合作真是太好了。
6. **prototype** (n.) — 原型、樣本
   - *We're building a prototype for our new IoT device.*
   - 我們正在為新的物聯網裝置打造一個原型。
7. **efficient** (adj.) — 效率高的
   - *His new workflow is really efficient and saves time.*
   - 他的新工作流程非常有效率且節省時間。
8. **sustainable** (adj.) — 永續的
   - *Many companies are now focusing on sustainable energy solutions.*
   - 許多公司現在都專注於永續能源解決方案。
9. **feedback** (n.) — 回饋、意見
   - *Don't forget to give your feedback on the new design.*
   - 別忘了對新設計提供你的回饋意見。
10. **update** (v.) — 更新
   - *Remember to update the system tonight before you leave.*
   - 記得今晚離開前更新系統喔。

## 📘 英文文法
### 未來式 (will 與 be going to)
Will 多用於即時決定或非計畫性的預測；be going to 則用於有計畫或基於現有證據的預測。
- I will call you later to discuss the project. (我稍後會打給你討論專案。)
- We are going to launch the new product next month. (我們下個月將會推出新產品。)
- Look at those dark clouds! It's going to rain soon. (看那些烏雲！馬上就要下雨了。)
### 片語動詞 (Phrasal Verbs)
動詞與介系詞或副詞組合成新意義，常用於口語中，理解上下文有助於掌握意思。
- I need to figure out how to fix this bug. (我需要弄清楚如何修復這個錯誤。)
- Let's set up a meeting for next Tuesday. (我們下週二安排一個會議吧。)
- Could you look into the customer's complaint? (你能調查一下客戶的投訴嗎？)

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - 毎日仕事に行きます。
   - 我每天都去工作。
2. **べんきょう** / 勉強 / benkyou — 學習
   - 私は日本語を勉強します。
   - 我學習日文。
3. **じかん** / 時間 / jikan — 時間
   - 時間がありますか。
   - 你有時間嗎？
4. **やすみ** / 休み / yasumi — 休息、假日
   - 今日は休みです。
   - 今天是休息日。
5. **つくる** / 作る / tsukuru — 製作、做
   - 新しいIoTデバイスを作ります。
   - 我製作新的物聯網裝置。
6. **つかう** / 使う / tsukau — 使用
   - AIツールを使います。
   - 我使用AI工具。
7. **まいにち** / 毎日 / mainichi — 每天
   - 毎日コーヒーを飲みます。
   - 我每天都喝咖啡。
8. **わかる** / 分かる / wakaru — 懂、明白
   - 日本語が分かりますか。
   - 你懂日文嗎？
9. **きょう** / 今日 / kyou — 今天
   - 今日は何曜日ですか。
   - 今天星期幾？
10. **かいぎ** / 会議 / kaigi — 會議
   - 明日、会議があります。
   - 明天有會議。

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です (是.../是...)
**結構**：名詞 + は + 名詞 + です
用來表示句子的主題或陳述身分、狀態等，は讀作「wa」，將主語與其後的陳述連結起來。
- これは私のパソコンです。(這是我的電腦。)
- 田中さんはエンジニアです。(田中先生是工程師。)
### 動詞的ます形 (肯定、否定、疑問)
**結構**：動詞ます形 (例：行きます / 行きません / 行きますか)
動詞ます形是用來表示禮貌和現在或未來動作，其否定形為「ません」，疑問形為「ますか」。
- 明日、仕事をします。(明天我會工作。)
- 今日は働きません。(今天我不工作。)
