# 2026-05-05（週二）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **implement** (v.) — 執行、實施（計畫或政策）
   - *We need to implement the new security features next week.*
   - 我們下週需要實施新的安全功能。
2. **optimize** (v.) — 優化、最佳化（系統或流程）
   - *Can you optimize this code for better performance?*
   - 你能優化這段程式碼以提升效能嗎？
3. **troubleshoot** (v.) — 排除故障、解決問題
   - *I spent an hour trying to troubleshoot my internet connection.*
   - 我花了一小時試圖排除我的網路連線問題。
4. **scenario** (n.) — 情境、情況（可能發生的狀況）
   - *Let's think about a few different usage scenarios for this device.*
   - 我們來想想這裝置的幾種不同使用情境吧。
5. **integrate** (v.) — 整合、結合（不同部分）
   - *It's crucial to integrate the new module seamlessly.*
   - 無縫整合新模組是至關重要的。
6. **prototype** (n.) — 原型、樣品（產品初步版本）
   - *The team is excited to show off the first prototype soon.*
   - 團隊很興奮能很快展示第一個原型機。
7. **efficient** (adj.) — 效率高的、有效率的
   - *Using an AI tool can make our workflow much more efficient.*
   - 使用AI工具能讓我們的工作流程更有效率。
8. **sustainable** (adj.) — 永續的、可持續的
   - *Renewable energy is key to a sustainable future.*
   - 再生能源是實現永續未來的關鍵。
9. **feedback** (n.) — 回饋意見、反應
   - *We always appreciate user feedback to improve our products.*
   - 我們總是感謝使用者回饋以改進我們的產品。
10. **monitor** (v.) — 監控、監測（系統或情況）
   - *We need to monitor the system's performance closely.*
   - 我們需要密切監控系統的效能。

## 📘 英文文法
### Conditional Sentences (Type 1)
用於表達未來可能發生或真實存在的條件與結果，句型為 If + 現在簡單式, will + 原形動詞。
- If you finish early, we can grab coffee. (如果你提早結束，我們可以去喝咖啡。)
- If the system crashes, I will restart it. (如果系統當機，我會重啟它。)
- If we optimize the code, it will run faster. (如果我們優化程式碼，它會跑得更快。)
### Phrasal Verbs (片語動詞)
動詞與介係詞或副詞結合，產生新的意思，常讓語氣更自然，在日常和職場對話中很常用。
- Could you help me set up the new server? (你能幫我設定新的伺服器嗎？)
- Don't forget to turn off the lights before you leave. (離開前別忘了關燈。)
- I can't figure out why this error keeps happening. (我搞不懂為什麼這個錯誤一直發生。)

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - 今日は仕事がとても忙しかったです。
   - 今天工作非常忙。
2. **べんきょう** / 勉強 / benkyou — 學習
   - 毎日日本語を勉強しています。
   - 我每天都在學習日語。
3. **でんき** / 電気 / denki — 電、電燈
   - 電気を消してください。
   - 請關燈。
4. **つかう** / 使う / tsukau — 使用
   - このAIツールをよく使います。
   - 我經常使用這個AI工具。
5. **ぎじゅつ** / 技術 / gijutsu — 技術
   - 最新の技術に興味があります。
   - 我對最新的技術很感興趣。
6. **じかん** / 時間 / jikan — 時間
   - 休憩する時間がほしいです。
   - 我想要休息時間。
7. **まいにち** / 毎日 / mainichi — 每天
   - 毎日コーヒーを飲みます。
   - 我每天喝咖啡。
8. **はじめる** / 始める / hajimeru — 開始
   - 会議は１０時に始まります。
   - 會議在10點開始。
9. **わかる** / 分かる / wakaru — 懂、明白
   - 先生の説明がよく分かりました。
   - 我非常明白老師的解釋。
10. **べんり** / 便利 / benri — 方便
   - このアプリはとても便利です。
   - 這個應用程式非常方便。

## 🗾 日文文法（N5）
### 〜ます / 〜ません / 〜ました / 〜ませんでした (Masu Form)
**結構**：動詞(ます形)
用於表達禮貌的肯定、否定、過去肯定和過去否定，是日語最基礎的動詞形式。
- 私は毎日コーヒーを飲みます。(我每天喝咖啡。)
- 昨日、日本語を勉強しませんでした。(我昨天沒有學習日語。)
### 名詞 + は + 名詞 + です / じゃありません
**結構**：名詞１ + は + 名詞２ + です / じゃありません
用於說明「名詞1是名詞2」或「名詞1不是名詞2」，是表達事物基本屬性的句型。
- これは私の本です。(這是我的書。)
- あれは私のパソコンじゃありません。(那個不是我的電腦。)
