## 📘 英文文法
### 動詞片語 (Phrasal Verbs)
動詞與介系詞或副詞結合，形成新意思的動詞片語，常用於口語，使表達更自然。
- I need to set up my new monitor. (我需要安裝我的新螢幕。)
- Did you come up with any good ideas? (你有想到什麼好點子嗎？)
- Let's catch up sometime next week. (我們下週找時間聊聊吧。)
### 條件句 (Conditional Sentences Type 1)
用於表達現在或未來可能發生且結果也可能實現的狀況，結構為 If + 現在式, will + 動詞原形。
- If it rains tomorrow, we will stay home. (如果明天會下雨，我們就待在家。)
- If you finish your tasks, you can go home early. (如果你完成任務，就可以早點回家。)
- I'll buy a new gadget if it's on sale. (如果它特價，我就會買一個新裝置。)

## 🗾 日文文法（N5）
### 動詞「ます形」 (Present/Past, Affirmative/Negative)
**結構**：動詞(ます形)
用來表示現在或未來的動作、習慣、意願，以及過去發生的動作或狀態，是禮貌體基本用法。
- わたしはまいにちコーヒーをのみます。 (我每天喝咖啡。)
- きのう、えいがをみませんでした。 (昨天我沒有看電影。)
### 名詞 + は + 名詞 + です / じゃありません (N1是N2 / N1不是N2)
**結構**：N1 は N2 です / N1 は N2 じゃありません
用於說明或否定名詞之間的關係，是日文最基本的肯定與否定句型，用來介紹或描述事物。
- わたしはエンジニアです。 (我是工程師。)
- これはわたしのほんじゃありません。 (這不是我的書。)
