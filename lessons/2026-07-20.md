# 2026-07-20（週一）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **optimize** (v.) — 優化/最佳化（性能）
   - *We need to optimize this code for better speed.*
   - 我們需要優化這段程式碼以提升速度。
2. **interface** (n.) — 介面；連接點
   - *The user interface is quite intuitive, right?*
   - 這個使用者介面蠻直觀的，對吧？
3. **troubleshoot** (v.) — 解決問題；故障排除
   - *Can you help me troubleshoot this Wi-Fi issue?*
   - 你能幫我排除這個Wi-Fi問題嗎？
4. **scenario** (n.) — 情境；設想狀況
   - *Let's consider a few user scenarios for the app.*
   - 我們來設想幾個應用程式的使用者情境。
5. **efficient** (adj.) — 效率高的；有效率的
   - *This new system is super efficient and saves time.*
   - 這個新系統超有效率，省下不少時間。
6. **prototype** (n.) — 原型；初步模型
   - *We're building a prototype for the IoT device.*
   - 我們正在為物聯網設備製作原型。
7. **commute** (n./v.) — 通勤；通勤路程
   - *My commute to work is pretty long, sadly.*
   - 我上班的通勤時間蠻長的，可惜。
8. **deadline** (n.) — 截止日期；最後期限
   - *We have a tight deadline for this project, let's go!*
   - 這個專案的截止日期很趕，加油！
9. **feedback** (n.) — 回饋；意見反應
   - *Please give us some feedback on the new design.*
   - 請給我們一些關於新設計的回饋意見。
10. **sustainable** (adj.) — 永續的；可持續的
   - *Renewable energy is a sustainable solution for the future.*
   - 再生能源是未來永續的解決方案。

## 📘 英文文法
### 現在完成式 (Present Perfect)
表達過去發生但影響持續到現在，或在未明確時間點完成的動作。
- I have worked on this project for three months. (我已經在這個專案上工作三個月了。)
- She has learned a lot about AI tools. (她學了很多關於AI工具的知識。)
- Have you ever optimized an embedded system? (你曾經優化過嵌入式系統嗎？)
### 條件句 (Conditional Sentences) - Type 1
用於表達未來可能發生且結果真實的情況，通常結構為「If + 現在簡單式, will + 原形動詞」。
- If the code works, we will deploy it next week. (如果程式碼成功，我們下週就會部署。)
- If you need help, just let me know. (如果你需要幫忙，就讓我知道。)
- If it rains tomorrow, I'll work from home. (如果明天會下雨，我就在家工作。)

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です
**結構**：名詞1 + は + 名詞2 + です
這是日語中最基本的肯定句型，用於表明或介紹事物，N1是主題，N2是其說明。
- 私はエンジニアです。(我是工程師。)
- これは私のパソコンです。(這是我的筆記型電腦。)
### 動詞(ます形) + ましょう
**結構**：動詞(ます) + ましょう
用於提議或邀請對方一起做某事，也可用於表示「讓我們...」，語氣輕鬆。
- 一緒に休憩しましょう。(我們一起休息吧。)
- 新しい技術（ぎじゅつ）を勉強しましょう。(我們來學習新技術吧。)
