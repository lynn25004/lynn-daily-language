# 2026-07-07（週二）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **implement** (v.) — 實施、執行計畫或功能
   - *We need to implement this new feature quickly.*
   - 我們需要快速實施這個新功能。
2. **optimize** (v.) — 優化、最佳化系統或程式
   - *Let's optimize the code for better performance.*
   - 我們來優化程式碼以提高效能。
3. **troubleshoot** (v.) — 排除故障、解決技術問題
   - *Can you help me troubleshoot this network issue?*
   - 你能幫我解決這個網路問題嗎？
4. **monitor** (v./n.) — 監控、監視狀態或數據
   - *We need to monitor the system's status closely.*
   - 我們需要密切監控系統狀態。
5. **integrate** (v.) — 整合、結合不同部分
   - *This new module integrates well with the system.*
   - 這個新模組與現有系統整合得很好。
6. **prototype** (n.) — 原型、初版模型
   - *The team is building a new prototype now.*
   - 團隊正在製作一個新原型。
7. **debug** (v.) — 除錯、找出並修正錯誤
   - *I spent hours debugging this small error.*
   - 我花了幾個小時才把這個小錯誤除掉。
8. **collaborate** (v.) — 合作、協同工作
   - *We need to collaborate more on this project.*
   - 我們需要在這個專案上多合作。
9. **deadline** (n.) — 截止日期、最後期限
   - *The deadline for this task is tomorrow.*
   - 這項任務的截止日期是明天。
10. **feedback** (n.) — 回饋、意見
   - *Please give us your feedback on the new design.*
   - 請給我們對新設計的回饋意見。

## 📘 英文文法
### 被動語態 (Passive Voice)
強調動作的承受者，而非執行者，常用於描述事實、程序或當執行者不重要時。
- The new system was developed last month. (新系統上個月開發完成。)
- This bug has been fixed by the team. (這個錯誤已經被團隊修復了。)
- All data must be backed up daily. (所有資料必須每天備份。)
### 條件句 - If-clause Type 1 (未來真實)
描述未來可能發生的情況及其結果，表示如果某事發生，另一事將會發生。
- If the sensor works, we will proceed with the test. (如果感測器能運作，我們就繼續測試。)
- If you finish coding early, we can grab coffee. (如果你提早寫完程式，我們可以去喝咖啡。)
- I will call you if I need any help with the setup. (如果我在設定上有任何需要，我會打電話給你。)

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です
**結構**：名詞1 + は + 名詞2 + です
用於介紹或說明一個名詞是另一個名詞，是日語最基本的肯定句型，通常用來定義或陳述事實。
- これはわたしのパソコンです。(這是我的電腦。)
- かれはエンジニアです。(他是工程師。)
### 動詞(ます形) + ましょう
**結構**：動詞(ます形) + ましょう
表示「一起做...吧」或「我們來做...吧」，用於提出建議或邀請對方共同行動。
- いっしょにプロトタイプをつくりましょう。(我們一起來製作原型吧。)
- きゅうけいしましょう。(我們休息一下吧。)
