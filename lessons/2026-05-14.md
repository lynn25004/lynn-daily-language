# 2026-05-14（週四）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **optimize** (v.) — 優化，使最佳化
   - *Let's optimize this code to run faster.*
   - 我們來優化這段程式碼讓它跑更快。
2. **troubleshoot** (v.) — 排除故障
   - *Can you help me troubleshoot this software bug?*
   - 你可以幫我排除這個軟體錯誤嗎？
3. **prototype** (n./v.) — 原型；製作原型
   - *We're building a prototype for our new product.*
   - 我們正在為我們的新產品製作一個原型。
4. **integrate** (v.) — 整合，結合
   - *It's hard to integrate new features seamlessly sometimes.*
   - 有時候要無縫整合新功能很難。
5. **efficient** (adj.) — 效率高的，有效率的
   - *Working efficiently helps us finish tasks on time.*
   - 有效率地工作幫助我們準時完成任務。
6. **commute** (v./n.) — 通勤
   - *How long is your daily commute to the office?*
   - 你每天通勤到辦公室要多久？
7. **deadline** (n.) — 截止日期
   - *Don't miss the deadline for submitting your report.*
   - 不要錯過提交報告的截止日期喔。
8. **prioritize** (v.) — 優先處理
   - *I need to prioritize my tasks for this week.*
   - 我需要為這週的任務排定優先順序。
9. **innovative** (adj.) — 創新的
   - *I love learning about innovative renewable energy tech.*
   - 我喜歡學習創新的再生能源技術。
10. **collaborate** (v.) — 合作
   - *Let's collaborate on this project to get it done.*
   - 我們一起合作這個專案把它完成吧。

## 📘 英文文法
### 條件句 (Conditional Sentences, Type 1)
用於描述現在或未來可能發生的情況及其結果，結構為 "If + S + V(現在式), S + will/can/may + V(原形)"。
- If it rains tomorrow, we will stay home and watch movies. (如果明天下雨，我們就待在家看電影。)
- If you finish your work early, you can join us for dinner. (如果你早點完成工作，就可以和我們一起吃晚餐。)
- If I learn new AI tools, my work might become more efficient. (如果我學新 AI 工具，我的工作可能會更有效率。)
### 片語動詞：與 "up" 相關
許多動詞加上 "up" 可以形成片語動詞，表達完成、增加、向上等意思，需多加練習。
- I need to set up my new IoT device tonight. (我今晚需要設定我的新物聯網裝置。)
- We should back up important data regularly to avoid losing it. (我們應該定期備份重要資料，以避免遺失。)
- Let's look up the meaning of this word if we don't know it. (如果我們不知道這個單字的意思，我們就查一下吧。)

## 🇯🇵 日文單字（N5）
1. **まいにち** / 毎日 / mainichi — 每天
   - 毎日日本語を勉強しています。
   - 我每天都在學習日文。
2. **しごと** / 仕事 / shigoto — 工作
   - 仕事は楽しいですが、少し忙しいです。
   - 工作很有趣，但是有點忙。
3. **べんきょう** / 勉強 / benkyou — 學習
   - 今日の夜は日本語の勉強をします。
   - 我今天晚上要學習日文。
4. **こんしゅう** / 今週 / konshuu — 這週
   - 今週の週末は何をしますか？
   - 你這週末要做什麼？
5. **たのしい** / 楽しい / tanoshii — 快樂的，有趣的
   - このゲームはとても楽しいです。
   - 這個遊戲非常有趣。
6. **じかん** / 時間 / jikan — 時間
   - 休憩する時間がありますか？
   - 你有時間休息嗎？
7. **でんしゃ** / 電車 / densha — 電車
   - 毎日電車で会社に行きます。
   - 我每天搭電車去公司。
8. **つかう** / 使う / tsukau — 使用
   - 新しいAIツールを使いたいです。
   - 我想要使用新的AI工具。
9. **わかる** / 分かる / wakaru — 懂，明白
   - 先生、もう分かりました。
   - 老師，我已經明白了。
10. **すこし** / 少し / sukoshi — 一點點
   - 少しだけ日本語が話せます。
   - 我只會說一點點日文。

## 🗾 日文文法（N5）
### 動詞(ます形) + たいです
**結構**：動詞(ます形) + たいです
用於表達說話者想做某事的願望或意圖，是日常對話中很常用的一個句型。
- 日本語を勉強したいです。 (我想學習日文。)
- 新しいAIツールを使いたいです。 (我想要使用新的AI工具。)
### 名詞 + が + あります / います
**結構**：名詞 + が + あります (非生物) / います (生物)
用於表達某物或某人存在，區分對象是生物（人、動物）或非生物（物品、地點）。
- 私の部屋にパソコンがあります。 (我的房間裡有一台電腦。)
- 会社に新しいエンジニアがいます。 (公司裡有一位新工程師。)
