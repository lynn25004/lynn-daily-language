# 2026-07-06（週一）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **implement** (v.) — 執行或實施某計畫
   - *Let's implement this new feature by Friday.*
   - 我們週五前實施這個新功能吧。
2. **optimize** (v.) — 優化效能或最佳化
   - *We need to optimize the code for better performance.*
   - 我們需要優化程式碼以提升效能。
3. **troubleshoot** (v.) — 找出並解決問題
   - *Can you help me troubleshoot this network issue?*
   - 你能幫我解決這個網路問題嗎？
4. **integrate** (v.) — 將不同部分整合
   - *We need to integrate the new sensor with the existing system.*
   - 我們需要將新感測器與現有系統整合。
5. **prototype** (n.) — 產品初步原型或雛形
   - *The team is building a new IoT prototype.*
   - 團隊正在建立一個新的物聯網原型。
6. **efficient** (adj.) — 做事快速且有效率
   - *This new algorithm is very efficient.*
   - 這個新演算法非常有效率。
7. **reliable** (adj.) — 可靠的，值得信任的
   - *We need a reliable power supply for the device.*
   - 我們的設備需要可靠的電源供應。
8. **collaborate** (v.) — 共同合作完成任務
   - *Let's collaborate on this challenging project.*
   - 我們一起合作這個有挑戰的專案吧。
9. **feedback** (n.) — 針對某事給予意見
   - *Your feedback on the design is very helpful.*
   - 你對設計的回饋非常有用。
10. **deadline** (n.) — 必須完成的最終日期
   - *The project deadline is approaching fast.*
   - 專案截止日期快到了。

## 📘 英文文法
### 條件句 (Conditional Sentences Type 1)
用於表達未來可能發生的情況及其結果，通常表示高機率會實現。
- If it rains, we will stay indoors.（如果下雨，我們就待在室內。）
- If you finish early, you can go home.（如果你早點完成，就可以回家了。）
- If the system crashes, we will fix it immediately.（如果系統當機，我們會立刻修復。）
### 動詞片語：Phrasal Verbs with "up"
動詞與介係詞"up"結合，產生不同於原動詞的新意義，常用於口語對話。
- I need to look up some info about renewable energy.（我需要查找一些關於再生能源的資訊。）
- Let's set up the new IoT device tomorrow.（我們明天來設定這個新的物聯網裝置吧。）
- Please back up your files regularly.（請定期備份你的檔案。）

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - きょうのしごとはおわりましたか。
   - 今天的工作完成了嗎？
2. **かいしゃ** / 会社 / kaisha — 公司
   - わたしのかいしゃはとおいです。
   - 我的公司很遠。
3. **でんき** / 電気 / denki — 電、電力
   - でんきをつけましょう。
   - 我們開燈吧。
4. **つくる** / 作る / tsukuru — 製作、製造
   - あたらしいシステムをつくります。
   - 我會製作一個新系統。
5. **まなぶ** / 学ぶ / manabu — 學習
   - わたしはにほんごをまなんでいます。
   - 我正在學習日文。
6. **わかる** / 分かる / wakaru — 懂、明白
   - これはわかりますか。
   - 這個你明白嗎？
7. **つかう** / 使う / tsukau — 使用
   - このツールをつかいます。
   - 我會使用這個工具。
8. **むずかしい** / 難しい / muzukashii — 困難的、難的
   - このもんだいはむずかしいです。
   - 這個問題很難。
9. **かいぎ** / 会議 / kaigi — 會議
   - きょうかいぎがあります。
   - 今天有會議。
10. **はじめる** / 始める / hajimeru — 開始
   - しごとをはじめる。
   - 開始工作。

## 🗾 日文文法（N5）
### 動詞て形 + ください
**結構**：動詞て形 + ください
用於禮貌地請求或指示對方做某事，是日常生活中非常實用的指令句型。
- まってください。（請稍等。）
- このほんをよんでください。（請讀這本書。）
### 名詞 + が あります/います
**結構**：名詞 + が あります / います
「あります」用於非生物，而「います」用於生物，表示某物或某人的存在。
- へやにつくえがあります。（房間裡有桌子。）
- オフィスにエンジニアがいます。（辦公室裡有工程師。）
