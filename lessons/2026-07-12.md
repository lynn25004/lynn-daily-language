## 📘 英文文法
### 條件句 (Conditional Sentences) Type 1
表達未來可能發生的真實情況，或在某條件下會發生的結果，通常用於計畫或預測。
- If it rains tomorrow, I will stay home. (如果明天會下雨，我就待在家。)
- If you finish your work early, we can go for a walk. (如果你早點完成工作，我們可以去散步。)
- If the system crashes, we'll troubleshoot it immediately. (如果系統當機，我們會立刻排除故障。)
### 片語動詞：put off / put together
'put off' 指延後某事；'put together' 指組合或組織某物，兩者皆為常用片語。
- Don't put off your important tasks until the last minute. (不要把重要任務拖到最後一刻。)
- I need to put together a presentation for next week. (我需要為下週準備一份簡報。)
- He can put together an IoT device very quickly. (他可以很快地組裝一個物聯網裝置。)

## 🗾 日文文法（N5）
### て形 + ください (請做...)
**結構**：動詞て形 + ください
用來禮貌地請求或指示對方做某事，是很實用的日常表達，例如請別人幫忙。
- ちょっと待ってください。 (請等一下。)
- この本を読んでください。 (請讀這本書。)
### が あります / が います (有...)
**結構**：名詞 + が + あります (無生命物) / 名詞 + が + います (有生命物)
「あります」用於表示無生命物存在（如物品、事件），「います」用於表示有生命物存在（如人、動物）。
- 部屋にパソコンがあります。 (房間裡有電腦。)
- 会社に猫がいます。 (公司裡有貓。)
