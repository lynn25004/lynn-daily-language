# 2026-06-01（週一）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **optimize** (v.) — 優化、最佳化
   - *Let's optimize this code for better performance.*
   - 我們來優化這段程式碼，讓效能更好。
2. **integrate** (v.) — 整合、集結
   - *We need to integrate this module into the main system.*
   - 我們需要將這個模組整合到主系統中。
3. **troubleshoot** (v.) — 排除故障、解決問題
   - *Can you help me troubleshoot why this device isn't working?*
   - 你能幫我排查一下為什麼這個裝置沒反應嗎？
4. **deadline** (n.) — 截止日期
   - *The deadline for this project is next Friday.*
   - 這個專案的截止日期是下週五。
5. **commute** (v.) — 通勤、上下班
   - *I usually commute by scooter, it's faster.*
   - 我通常騎機車通勤，比較快。
6. **efficient** (adj.) — 高效率的
   - *We need a more efficient way to manage our tasks.*
   - 我們需要一個更有效率的方式來管理任務。
7. **prototype** (n.) — 原型、樣本
   - *The team is testing the first prototype of our new IoT device.*
   - 團隊正在測試我們新物聯網裝置的第一個原型。
8. **debug** (v.) — 除錯
   - *I spent all morning trying to debug this tricky issue.*
   - 我整個早上都在試圖除錯這個棘手的問題。
9. **feedback** (n.) — 回饋、意見
   - *We'll gather user feedback to improve our product.*
   - 我們會收集使用者回饋來改進產品。
10. **recharge** (v.) — 充電、休息
   - *After a busy week, I need some time to recharge.*
   - 忙碌一週後，我需要一些時間好好充電。

## 📘 英文文法
### 常用動詞片語：以 "up" 結尾的片語
"up" 常與動詞結合，構成具有「完成」、「增加」、「向上」、「出現」等不同涵義的動詞片語，讓對話更自然。
- Let's set up the new server this afternoon. (我們今天下午來安裝新的伺服器吧。)
- I need to catch up on my emails after the meeting. (會議後我得趕緊處理積壓的電子郵件。)
- Could you back up these files to the cloud? (你能把這些檔案備份到雲端嗎？)
### 條件句 (Conditional Type 1)
用於描述未來可能發生且結果也可能實現的狀況，通常結構為 "If + 現在簡單式, will/can + 動詞原形"。
- If we finish this early, we can go home on time. (如果我們提早完成，就能準時回家。)
- If the system crashes, I will restart it immediately. (如果系統當機，我會立刻重啟它。)
- If you have any questions, feel free to ask me. (如果你有任何問題，儘管問我。)

## 🗾 日文文法（N5）
### 〜たいです (想做...)
**結構**：動詞ます形 + たいです
表示說話者「想要做」某事的願望或希望，動詞去掉ます後加上たいです。
- 新しい IoT デバイスを作りたいです。(我想製作新的 IoT 裝置。)
- 週末はゆっくり休みたいです。(週末我想好好休息。)
### 動詞て形 + ください (請做...)
**結構**：動詞て形 + ください
用於向對方提出請求或指示，表示「請做某事」，語氣禮貌。
- このエラーを直してください。(請修正這個錯誤。)
- 日本のニュースを見てください。(請看日本新聞。)
