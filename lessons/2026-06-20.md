# 2026-06-20（週六）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **firmware** (n.) — 韌體，控制硬體的程式
   - *I'm updating the device's firmware now.*
   - 我現在正在更新裝置的韌體。
2. **debug** (v.) — 偵錯，找出並修復程式錯誤
   - *It took hours to debug this issue.*
   - 花了好幾個小時才把這個問題偵錯完。
3. **prototype** (n.) — 原型，初步模型或樣品
   - *We built a prototype for the new IoT device.*
   - 我們為新的物聯網裝置做了一個原型。
4. **optimize** (v.) — 優化，使其效能更好
   - *I need to optimize this code for speed.*
   - 我需要優化這段程式碼以提高速度。
5. **recharge** (v.) — 充電，恢復精力或電力
   - *I need to recharge myself this weekend!*
   - 我這個週末需要為自己充電！
6. **project** (n.) — 專案，計畫或任務
   - *How's your current project going?*
   - 你現在的專案進展如何？
7. **hobby** (n.) — 嗜好，閒暇時的活動
   - *What's your favorite weekend hobby?*
   - 你週末最喜歡的嗜好是什麼？
8. **explore** (v.) — 探索，發掘新事物
   - *I want to explore new AI tools.*
   - 我想要探索新的 AI 工具。
9. **solution** (n.) — 解決方案，問題的答案
   - *We need a quick solution for this bug.*
   - 我們需要一個快速解決這個錯誤的方案。
10. **update** (n.) — 更新，最新消息或動作
   - *Do you have any updates on the project?*
   - 你有關於這個專案的任何更新嗎？

## 📘 英文文法
### 使用 "Look forward to + V-ing"
表達期待未來會發生的事情或活動，後面要接動名詞 (V-ing)，不能接原型動詞。
- I'm looking forward to debugging this new feature. (我很期待為這個新功能偵錯。)
- Are you looking forward to the weekend? (你期待週末嗎？)
- She's looking forward to learning Japanese. (她很期待學習日文。)
### 情態助動詞 "Might" vs "May"
兩者都表示可能性，'might' 通常表示可能性較低或更不確定，'may' 則可能性較高。
- I might work on my embedded project later. (我等一下可能會做我的嵌入式專案。)
- It may rain, so bring an umbrella. (可能會下雨，所以帶把傘。)
- We might explore a new AI tool next week. (我們下週可能會探索一個新的 AI 工具。)

## 🇯🇵 日文單字（N5）
1. **しゅうまつ** / 週末 / shuumatsu — 週末
   - 週末は何をしますか？
   - 週末你做什麼？
2. **べんきょう** / 勉強 / benkyou — 學習
   - 毎日日本語を勉強します。
   - 我每天學習日文。
3. **しごと** / 仕事 / shigoto — 工作
   - 今日は仕事がありません。
   - 今天沒有工作。
4. **じかん** / 時間 / jikan — 時間
   - 自由な時間がありますか？
   - 你有空閒時間嗎？
5. **しゅみ** / 趣味 / shumi — 嗜好
   - 私の趣味はゲームです。
   - 我的嗜好是玩遊戲。
6. **つくる** / 作る / tsukuru — 製作
   - IoTデバイスを作りたいです。
   - 我想製作 IoT 裝置。
7. **みる** / 見る / miru — 看
   - 映画を見ますか？
   - 你看電影嗎？
8. **ともだち** / 友達 / tomodachi — 朋友
   - 友達と会います。
   - 我會見朋友。
9. **たのしい** / 楽しい / tanoshii — 快樂的、有趣的
   - 週末は楽しいです。
   - 週末很快樂。
10. **すこし** / 少し / sukoshi — 一點點
   - 日本語が少し話せます。
   - 我會說一點點日文。

## 🗾 日文文法（N5）
### 動詞(ます形) + たいです
**結構**：動詞(ます形) + たいです
表示「想要做某事」的願望或慾望，將動詞的ます形去掉ます，再接續 たいです。
- 私は日本語を勉強したいです。(Watashi wa Nihongo o benkyou shitai desu.) - 我想學習日文。
- 新しいAIツールを使いたいです。(Atarashii AI tsuuru o tsukaitai desu.) - 我想使用新的AI工具。
### 名詞 + は + (形容詞/名詞) + です
**結構**：名詞 + は + (形容詞/名詞) + です
這是最基本的肯定句型，表示「A是B」或「A是(形容詞)」，は是提示助詞。
- これは私のIoTプロジェクトです。(Kore wa watashi no IoT purojekuto desu.) - 這是我的IoT專案。
- 週末は楽しいです。(Shuumatsu wa tanoshii desu.) - 週末很快樂。
