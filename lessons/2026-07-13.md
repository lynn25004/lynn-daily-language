# 2026-07-13（週一）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **optimize** (v.) — 最佳化，讓效率更好
   - *We need to optimize this code for better performance.*
   - 我們需要最佳化這段程式碼以獲得更好的效能。
2. **implement** (v.) — 執行，實際應用
   - *Let's implement the new feature by next week.*
   - 我們下週前要實作這個新功能。
3. **debug** (v.) — 除錯，找出並修正問題
   - *I spent hours trying to debug this circuit!*
   - 我花了幾個小時試圖除錯這個電路！
4. **deadline** (n.) — 截止日期，期限
   - *The deadline for this project is really tight.*
   - 這個專案的截止日期真的非常緊迫。
5. **collaborate** (v.) — 合作，共同完成
   - *We should collaborate on this task to finish faster.*
   - 我們應該在這個任務上合作以更快完成。
6. **prioritize** (v.) — 排序，決定優先順序
   - *It's hard to prioritize tasks when everything is urgent.*
   - 當所有事情都很緊急時，很難決定任務的優先順序。
7. **prototype** (n.) — 原型，初步模型
   - *The team is building a new IoT prototype.*
   - 團隊正在製作一個新的物聯網原型。
8. **efficient** (adj.) — 有效率的，省時省力
   - *Using AI tools makes our workflow more efficient.*
   - 使用AI工具讓我們的工作流程更有效率。
9. **feedback** (n.) — 回饋，意見反應
   - *Could you give me some feedback on my design?*
   - 你可以給我一些關於我設計的回饋嗎？
10. **sync** (v.) — 同步，保持一致
   - *Let's sync up after lunch to discuss the progress.*
   - 我們午餐後再同步一下，討論進度。

## 📘 英文文法
### 未來式：be going to vs. will
"be going to" 表示事先計畫好的未來，而 "will" 則多用於臨時決定、預測或承諾，語氣較彈性。
- I am going to fix the bug this afternoon. (我今天下午會去修那個錯誤。)
- Oh, I'll help you with that right now! (喔，我現在馬上幫你！)
- I think it will rain tomorrow, so bring an umbrella. (我想明天會下雨，所以帶把傘吧。)
### 使用 "should" 提出建議或義務
"should" 用來給予建議、表達義務或提出看法，語氣比 "must" 輕鬆，暗示這是個好主意。
- You should take a break, you look tired. (你該休息一下了，你看起來很累。)
- Engineers should always back up their data. (工程師應該隨時備份他們的資料。)
- I think we should try a different approach for this problem. (我覺得我們應該用不同的方法來解決這個問題。)

## 🇯🇵 日文單字（N5）
1. **しごと** / 仕事 / shigoto — 工作
   - わたしのしごとはおもしろいです。
   - 我的工作很有趣。
2. **やすみ** / 休み / yasumi — 休息、假日
   - きょうはやすみです。
   - 今天是假日。
3. **きょう** / 今日 / kyou — 今天
   - きょう、なにをしますか。
   - 今天要做什麼呢？
4. **あした** / 明日 / ashita — 明天
   - あした、かいしゃへいきます。
   - 我明天要去公司。
5. **まいにち** / 毎日 / mainichi — 每天
   - まいにち、にほんごをべんきょうします。
   - 我每天都學習日文。
6. **かいしゃ** / 会社 / kaisha — 公司
   - あなたのかいしゃはどこですか。
   - 你的公司在哪裡？
7. **つくえ** / 机 / tsukue — 書桌
   - つくえのうえにほんです。
   - 書桌上有一本書。
8. **いそがしい** / 忙しい / isogashii — 忙碌的
   - こんしゅうはいそがしいです。
   - 這週很忙碌。
9. **はやく** / 早く / hayaku — 快點、早點
   - はやくきてください。
   - 請早點來。
10. **おわります** / 終わります / owarimasu — 結束（動詞ます形）
   - かいぎはすぐおわります。
   - 會議馬上就要結束了。

## 🗾 日文文法（N5）
### 名詞 + は + 名詞 + です
**結構**：名詞1 + は + 名詞2 + です
這是日文最基本的肯定句型，用於介紹或說明事物，表達「A是B」的意思，助詞「は」讀作「wa」。
- これはペンです。(這是一支筆。)
- わたしはエンジニアです。(我是工程師。)
### 動詞ます形
**結構**：動詞 + ます
動詞ます形是日文動詞最基本的禮貌體，用於一般日常對話及肯定句，表達現在或未來動作。
- コーヒーをのみます。(我喝咖啡。)
- 毎日、日本語をべんきょうします。(我每天學習日文。)
