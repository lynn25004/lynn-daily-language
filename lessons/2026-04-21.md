# 2026-04-21（週二）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **firmware** (n.) — 控制硬體運作的軟體（韌體）
   - *We need to update the firmware on these IoT devices.*
   - 我們需要更新這些物聯網裝置的韌體。
2. **debug** (v.) — 找出並修正程式或系統錯誤
   - *I spent all morning trying to debug this code.*
   - 我整個早上都在試著為這段程式碼除錯。
3. **prototype** (n.) — 產品或系統的初步模型
   - *The team is building a new prototype for the smart home system.*
   - 團隊正在為智慧家庭系統建造一個新原型。
4. **optimize** (v.) — 使某物達到最佳效果
   - *How can we optimize battery life for our embedded device?*
   - 我們如何能優化嵌入式裝置的電池續航力？
5. **challenge** (n.) — 困難或需要努力才能完成的任務
   - *Finishing this project on time will be a real challenge.*
   - 準時完成這個專案將會是一個真正的挑戰。
6. **collaborate** (v.) — 與他人一起工作或合作
   - *We need to collaborate closely with the software team.*
   - 我們需要和軟體團隊緊密合作。
7. **deadline** (n.) — 完成某事的最終期限
   - *Is the deadline for the report still next Friday?*
   - 報告的截止日期還是下週五嗎？
8. **innovative** (adj.) — 引入新方法、思想或產品的
   - *Your idea for the renewable energy system is very innovative.*
   - 你關於再生能源系統的想法非常創新。
9. **flexible** (adj.) — 可輕鬆改變或調整的
   - *Our work schedule is pretty flexible this week.*
   - 我們這週的工作行程很有彈性。
10. **prioritize** (v.) — 將事情按重要性排序
   - *I need to prioritize tasks for tomorrow morning.*
   - 我需要為明天早上優先處理任務。

## 📘 英文文法
### 動名詞 (Gerunds) 作為主詞或受詞
動名詞 (-ing 形式) 可以像名詞一樣在句子中充當主詞或受詞，表示一個動作或活動。
- Debugging code can be really frustrating sometimes. (除錯程式碼有時候真的很令人沮喪。)
- I enjoy learning about new AI tools and applications. (我喜歡學習新的 AI 工具和應用。)
- She suggested optimizing the system for better performance. (她建議優化系統以獲得更好的性能。)
### 未來完成式 (Future Perfect Tense)
表示在未來某個時間點之前，一個動作將會完成。結構是 "will have + 過去分詞"。
- By next Tuesday, I will have finished writing the report. (到下週二之前，我將會寫完這份報告。)
- They will have developed a new prototype by the end of the month. (他們將在月底前開發出一個新的原型。)
- You will have optimized the energy usage before the presentation. (在簡報之前，你將會優化能源使用。)

## 🇯🇵 日文單字（N5）
1. **おはようございます** / おはようございます / ohayou gozaimasu — 早上好
   - おはようございます！今日のお天気はどうですか？
   - 早上好！今天天氣如何？
2. **しごと** / 仕事 / shigoto — 工作
   - 私の仕事はエンジニアです。
   - 我的工作是工程師。
3. **べんきょう** / 勉強 / benkyou — 學習
   - 毎日日本語を勉強します。
   - 我每天學習日語。
4. **きゅうけい** / 休憩 / kyuukei — 休息
   - ちょっと休憩しませんか？
   - 要不要休息一下？
5. **じかん** / 時間 / jikan — 時間
   - 会議の時間は何時ですか？
   - 會議時間是幾點？
6. **つかう** / 使う / tsukau — 使用
   - このAIツールはとても便利です。
   - 這個 AI 工具非常好用。
7. **わかる** / わかる / wakaru — 明白／懂
   - 日本語が少しわかります。
   - 我懂一點日語。
8. **むずかしい** / 難しい / muzukashii — 困難的
   - この問題は少し難しいです。
   - 這個問題有點難。
9. **はじめる** / 始める / hajimeru — 開始
   - プロジェクトは来週から始めます。
   - 專案從下週開始。
10. **おわる** / 終わる / owaru — 結束
   - 今日の仕事はもうすぐ終わります。
   - 今天的工作很快就結束了。

## 🗾 日文文法（N5）
### ～は～です (wa...desu)
**結構**：名詞 + は + 名詞/形容詞 + です
用於介紹或說明主語的身份、狀態或性質，是最基本、禮貌的肯定句形式。
- 私は台湾人です。(我是台灣人。)
- これは私のパソコンです。(這是我的電腦。)
- 日本語は面白いです。(日語很有趣。)
### 動詞ます形 + たいです (tai desu)
**結構**：動詞ます形 + たいです
表示說話者想要做某事。把動詞的ます形去掉ます，加上たいです。
- 私は日本へ行きたいです。(我想要去日本。)
- 新しいAIツールを使いたいです。(我想要使用新的AI工具。)
