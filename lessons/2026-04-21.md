# 2026-04-21（週二）英日文每日課程

## 🇬🇧 英文單字（TOEIC 600-800）
1. **implement** (v.) — 執行、實施計畫或系統
   - *We need to implement the new security features by next month.*
   - 我們需要在下個月前實施新的安全功能。
2. **optimize** (v.) — 優化、最佳化以提高效能
   - *Can you help me optimize this code for better performance?*
   - 你可以幫我優化這段程式碼以提升效能嗎？
3. **troubleshoot** (v.) — 故障排除、解決問題
   - *I spent all morning trying to troubleshoot the network issue.*
   - 我花了一整個早上試圖排除網路問題。
4. **prototype** (n.) — 原型、樣本產品
   - *The team is building a new prototype for the smart home device.*
   - 團隊正在為智慧家庭裝置打造一個新原型。
5. **collaborate** (v.) — 合作、協作完成任務
   - *It's important for us to collaborate closely on this project.*
   - 對我們來說，在這個專案上密切合作很重要。
6. **feature** (n.) — 特色、功能
   - *The new software has a lot of cool features, like voice control.*
   - 這個新軟體有很多很酷的功能，像是語音控制。
7. **deadline** (n.) — 截止日期、最終期限
   - *Don't worry, we can definitely meet the deadline for the report.*
   - 別擔心，我們絕對能在報告截止日期前完成。
8. **innovative** (adj.) — 創新的、有新意的
   - *We're looking for innovative solutions to reduce energy consumption.*
   - 我們正在尋找創新的解決方案來減少能源消耗。
9. **scenario** (n.) — 情境、設想
   - *Let's consider a few different scenarios for the system failure.*
   - 讓我們考慮幾種不同的系統故障情境。
10. **update** (v.) — 更新、提供最新資訊
   - *Did you update the software on your embedded device?*
   - 你有更新你嵌入式裝置上的軟體嗎？

## 📘 英文文法
### Conditional Sentences (Type 1)
用於描述在特定條件下，未來很可能會發生的結果或情況。
- If it rains, we will stay home.（如果下雨，我們就待在家裡。）
- If you finish your tasks early, you can join the meeting.（如果你提早完成任務，就可以參加會議。）
- If the system works, we will launch it next week.（如果系統運作正常，我們下週就會啟動它。）
### Passive Voice (被動語態)
當動作的承受者比執行者更重要，或執行者不明時使用。
- The report was submitted yesterday.（這份報告昨天提交了。）
- New features are being developed by our team.（新功能正在由我們團隊開發中。）
- This IoT device is powered by renewable energy.（這個物聯網設備由再生能源供電。）

## 🇯🇵 日文單字（N5）
1. **おはよう** / ohayou — 早安
   - おはようございます！今日はいい天気ですね。
   - 早安！今天天氣真好呢。
2. **しごと** / 仕事 / shigoto — 工作
   - 私は毎日仕事をします。
   - 我每天都工作。
3. **あさ** / 朝 / asa — 早上
   - 毎朝コーヒーを飲みます。
   - 每天早上喝咖啡。
4. **かいしゃ** / 会社 / kaisha — 公司
   - 会社へ行きます。
   - 去公司。
5. **でんしゃ** / 電車 / densha — 電車
   - 毎日電車で会社に行きます。
   - 每天搭電車去公司。
6. **じかん** / 時間 / jikan — 時間
   - 今、何時ですか？
   - 現在幾點？
7. **かいぎ** / 会議 / kaigi — 會議
   - 午後から会議があります。
   - 下午有會議。
8. **べんきょう** / 勉強 / benkyou — 學習、讀書
   - 毎日日本語を勉強します。
   - 每天學習日文。
9. **たいへん** / 大変 / taihen — 辛苦、嚴重、不得了
   - この仕事は大変ですね。
   - 這份工作很辛苦呢。
10. **きゅうけい** / 休憩 / kyuukei — 休息
   - ちょっと休憩しましょう。
   - 稍微休息一下吧。

## 🗾 日文文法（N5）
### 「～は～です」
**結構**：名詞 + は + 名詞/形容詞 + です
此為最基本的肯定句型，用於說明句子的主題是什麼，或者其目前的狀態如何。
- 私はエンジニアです。（我是工程師。）
- これは私のパソコンです。（這是我的電腦。）
### 動詞「～ます/ません/ました/ませんでした」
**結構**：動詞(ます形)
動詞的禮貌體，用於表示現在、未來或過去的動作，以及肯定或否定。
- 私は毎日日本語を勉強します。（我每天學習日文。）
- 昨日、会社に行きませんでした。（昨天沒有去公司。）
